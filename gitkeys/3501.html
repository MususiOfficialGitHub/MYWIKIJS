Register

Use this module to register new employees.

To access this window, click: Human Resources Module -> Navigate -> Operations ->Employees -> Register.

 The following window will open up:

All the company's employees will be displayed. 

The employees are grouped according to the grouping filter selected ~ Branch in the case above.

To group them using a different item, select the preferred grouping item from the drop down as shown below: 

The grouping will be affected automatically.

 Registering A New Employee

 Click on the create button to open the following screen: 

NOTE:  For an individual to be created as an employee, he must first be registered as a customer in the system. 

Customer - Pick the customer to be captured as an employee from the Customer Look Up dialog box: 

 Designation -  Pick the employee's designation from the  Designation Lookup dialog box:  

 If the designation does not exist, set it up under Human Resources Module -> Navigate -> Setup - > Designations

 Branch - pick the branch to which the employee will be posted from the Branch Lookup dialog box:

 

 Department - Pick the employee's department from the Department Lookup dialog box:

 If the department does not exist, set it up under Human Resources Module -> Navigate -> Setup - > Departments.

Employee Type - Pick the employee's employee type from the Employee Lookup dialog box:

 Select the employee's Blood Group: 

 

The employee's P.I.N will be automatically displayed on this field. 

Type in the Employee's  N.S.S.F & N.H.I.F numbers.

Cost Center - Pick the Cost Center from the Cost center Lookup dialog box:

Click on 

to select Hire & Release Dates, alternatively you can type in the dates in the respective fields.

Printing Mode: Select a printing mode from the drop-down List as shown Below.

Inactivity timeout: Enter the time before system triggers user password in case of inactivity.

 Input Remarks accordingly.

System initialization and lock time: Specify the time a user is able to use the system.

 

Capture and add the employee's emergency contacts and dependents details accordingly and press on Add on each entry(as shown in the first highlight below).

 In case you want to delete an entry simply select an entry below (as indicated with an arrow) and click Remove

Notice the flags at the bottom of the screen:

 

  Check the  

  flag to Enforce Initialization/lock times in which the user can use the system.

Check the 

 

flag to enforce the use of biometrics for the employee's log in. 

Check the 

flag if the employee should receive notifications about operations carried out.

Check the 

flag to Enable user to use the default printer. 

 Check the 

flag to send pay slips to employees' emails upon payroll processing.

NOTE: Before you decide to check the biometrics

flag, the employee's prints must have been captured under the customer register, 

so that there is a verifiable fingerprint template. 

 Refer to Customer Register.

 

click Create button to create the employee into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.

Editing An Employee 

Identify the employee to be edited and right-click on it 

Click the edit icon 

against the employee to open the following window:

Make necessary changes to the employee's details.

Update to save the changes. 

On successful completion you will get a pop-up message confirming that the operation was successful.

Viewing An Employee's Details 

 Click on the view icon 

against the employee to open the following screen :

The employees' details will be populated as in the figure above.

The additional customer details are displayed in the subsequent tabs on the same window;

see the figure below:

  Individual Particulars tab - displays individual's address.

   Address Tab - displays the employee's addresses. 

Emergency contacts - displays emergency contacts

Dependents - displays dependents information.

    Referees Tab -  displays the employee's referees.

    Specimen  -  displays the employee's identity card- front, back, passport photo and signature. 

   Savings Account -  displays the employee's savings accounts

    Loan Accounts - displays the employee's loan accounts. 

    Investment Accounts - displays the employee's investment accounts.

    Disciplinary Cases - displays the employee's disciplinary cases, together with documents if available. 

    Trainings - displays the employee's trainings / seminars attended together with documents if available.
