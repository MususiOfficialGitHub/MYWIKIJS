          Employee Exit Verification. 


 

       Use this sub-module to verify an exit request for an exiting employee. 

To access this window, click : Human Resources Module -> Navigate -> Operations  -> Exit -> Verify. 


 

 The following screen will open up:


 

Employee Exit Verification Fig 1


 

The pending entries will be displayed. 


 

     To Verify An Employee Exit : 

     

            Click on the verification icon 

against the entry, see below:  

    

NOTE :  You can only verify an exit request whose status is pending. 


 

Employee Exit Verification Fig 2


 

   To open the following window:


 

 

Employee Exit Verification Employee Exit Verification  Fig 3


 

    Crosscheck the customer's details to ensure they are correct. 

Audit Remarks - Provide appropriate audit remarks in the audit remarks field then select the verify action from the drop down menu. 


 

            Click on the update button to save the information into the system. 

   

On successful completion you will get a pop up message confirming that the operation was successful.
