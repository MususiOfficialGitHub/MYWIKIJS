Cash Withdrawal Requests

Use this module to create, authorize or reject a cash withdrawal request(s). 

To access this module, Click on Front-Office Management -> Operations -> Treasury -> Cash Withdrawal Requests. 

The following screen will open up: 

 

Creating A Cash Withdrawal Request

To create a cash withdrawal request, click on the Add New button. 
NOTE: Any withdrawal request by the teller that above the maximum allowed limit will automatically place a request here.

The following screen will open:

Full Account Number - Select the Customers "Savings Account" by clicking on the Full Account Number lookup icon. 

The details of the selected account will be loaded.

Amount - Enter the Amount that the customer wants to place a notice for in the amount field.

Primary Description - Enter the Primary Description, for example, something to identify the transaction.

Secondary Description - Enter the Secondary Description, for example, something to identify the transaction.

Reference - Enter the Reference, this could be the cheque number or something to identify the transaction.

Remarks - Enter the Remarks accordingly.

Click on the create button to save changes.


 

Authorizing A Cash Withdrawal Request

To identify a withdrawal request to authorize use date range or the search field. 

Using date - select the date range or a custom date range when the withdrawal request was placed by selecting a start date and an end date.

Using the search field - use the search field to enter the details of the withdrawal request, i.e. by typing a customer's account number or name.

A withdrawal request has the authorization status as pending, rejected or authorized/paid.

After you have identified a withdrawal request pending approval, right click on the request and select the authorize option as shown in the figure below.

Remember: You can only authorize a request whose status is pending.

The following window will open:

Remarks - Enter appropriate remarks on the remarks field referring to the withdrawal request.

Select the Auth Option as authorize on the dropdown menu.

 

Click on the update button to complete the authorization process. A confirmation pop up message will appear confirming that the operation was completed successfully. 

NOTE: This module is also used for the authorization of cash payments made by the teller under Sundry Receipts/Payments -> Cash Payment.
Refer to   Teller -> Sundry Receipts Payments -> Cash Payment

 

 

View A Cash Withdrawal Request

To identify a withdrawal request to view, use date range or the search field. 

Using date - select the date range or a custom date range when the withdrawal request was placed by selecting a start date and an end date.

Using the search field - use the search field to enter the details of the withdrawal request, i.e. by typing a customer's account number or name.

A withdrawal request has the authorization status as pending, rejected or authorized/paid.

After you have identified a withdrawal request, right click on the request and select the view option. 

The following window will open:

The Customer Account Statement for the member will be shown.
