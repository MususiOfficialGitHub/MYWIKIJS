Emolument Types

Use this sub-module to set up external agencies and link them to a GL account and Branch.

To access this window, click: Front Office Management - > Setup - > Emolument Types

The following screen will open up:


 

Creating an Emolument Type: 

Click on the add Button, to open the following screen:

Name - Enter the name  of the Treasury (probably by branch name)

G/L Account - Click on the lookup icon to select the appropriate G/L Account from the G/L Account Lookup dialog box: 

Amount - Enter the amount to be apportioned to each customer that will be paid using this emolument type.(does not mean the total amount).

Transaction Ownership - Click on the Transaction Ownership drop down to select a Transaction Ownership type.

Auto wire Medium - Click on the Auto wire Medium drop down to select an Auto wire Medium type.

Applicable Charges Tab: 

Select the appropriate charges for that particular Emolument Type. 

Beneficiary Products Tab: 

Check the product(s) to receive the amount specified for the Emolument Type. 

Flags Tab : 

Auto wire? - Click the flag when the External Agency is no longer being used

Is Locked? - Click the flag when the External Agency is no longer being used

When done click on the CREATE Button to complete this task.

On successful completion you will get a pop up message confirming that the operation was successful.

                                           
 

Editing an Emolument Type: 

First identify the Emolument Type to be edited right click on it: 

 Click on the edit icon against it to open the following screen: 

On the above screen, make the necessary changes to the Name, G/L Account, Amount, Transaction Ownership, Auto wire Medium, Applicable Charges, Beneficiary Products and Flags.

When done click on the UPDATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.
