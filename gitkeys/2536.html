Catalogue

Use this sub module to view a listing/catalog of all running, paid/matured, revoked Fixed Deposits Accounts.

To Access this sub module, Click on Front-Office -> Operations -> Fixed Deposits -> Catalogue.

The following screen will open up:

From the above screen you will be able to see the status of all fixed deposits whether Running, Revoked or paid.

Select the appropriate status to be viewed then refresh to view the fixed deposit listing of the selected status. 

Other details like the maturity date, expected interest, total expected, customer name & account and the date paid are also displayed. 

Select the number of items you want to view per page then refresh to reload data to the screen.
