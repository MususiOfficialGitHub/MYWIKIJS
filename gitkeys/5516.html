Verification

 Use this module to Verify loan restructuring requests.

To access this module: Credit Management -> Navigate -> Operations -> Loan Restructuring->   Verification

 The following window will open up: 

Verifying a Request.

Click on the look up and select an entry previously created :

NOTE : The verifying officer will make an assessment of the entry and based on that he/she will enter a reference for the verification and some remarks.
