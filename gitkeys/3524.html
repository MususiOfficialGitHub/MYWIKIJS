History Management

 Use this module to carry out history management for the different leave types.

To access this window, click: Human Resources Module -> Navigate -> Operations -> Leave -> History Management. 

The following Window will open up : 

  Employee - Look up the employee whose history is to be affected.

 Leave type - Look up the leave type to effect history for.

On the management tab as shown below:

 Define the number of days, adjustment type and reference for the action.

 Click the Update button to effect the changes into the system.

On successful completion you will get a pop-up message confirming that the operation was successful.
