Refund Types

Use this sub module to capture the refund types and their applicable charges if any.

To access this module: Credit Management -> Setup -> Refund Types.

The following screen will open up: 

 

 

Creating Loan Refund Type

To  create a refund type, click on the Add New button to open the following window: 

On the above screen: 

Name - Type the name of the refund type.

Transaction Ownership - Select the appropriate type of transaction ownership from the drop down list.

Applicable Charges: Tick the appropriate charges from the list of charges. 

Recover Sundry Arrearages? - Select this flag to recover loan arrears on refund processing.

Is Locked? - Select this flag to lock the refund that it may no longer be used.

 

On successful completion you will get a pop up message confirming that the operation was successful.


 

Editing A Refund Type.

Identify the refund type to be edited and right click on the on the refund type to select the edit option against the type. The following screen will open up: 

Make the necessary changes to the refund type then click on the update button to save the changes into the system. 

Recover Sundry Arrearages? - Select this flag to recover loan arrears on refund processing.

Is Locked? - Select this flag to lock the refund that it may no longer be used.

 

On successful completion you will get a pop up message confirming that the operation was successful.
