Salary Periods 


 

               Use this sub module to create a salary period(s). This is just the month for which the salary ids being processed.

To access this module click on Human Resource Module -> Navigate -> Operations-> Salary -> Salary Periods. 


 

            The following screen will open up:


 

Salary Periods Fig 1.


 

                       The available salary periods are displayed. 


 

Creating A Salary Period.


 

 Click on the create button to open the following screen :


 

Salary Periods Fig 2.


 

            On the above screen :


 

Posting Period - Click on the Look up icon 

to Select the posting period from the  Posting Period Lookup Dialog Box


 

       Reference Month - Select the appropriate month from the drop down menu. 


 

   Employee Type -  Click on the Look up icon 

to Select  the appropriate employee Type  from the Employee Type Dialog Box


 

    Define the period's Remarks accordingly.


 

Value date - Click 

to select date.( Alternatively you can type in the date)


 

                                     Click  create  to capture the period into the system.

On successful completion you will get a pop up message confirming that the operation was successful.


 


 

                         


 


 

Editing A Salary Period :


 

           Identify the salary period to be edited 

                 Click on the edit icon 

against the salary period to open the following window : 


 

Salary Periods Fig 4.  


 

        On the above screen :

           

              Make applicable  changes to the fields to be affected.

               

       Check the above flag if the salary being processed is for a past month but is being paid in the succeeding month.

      e.g If the salary being paid is for the Month of May but is being processed in June, check the Enforce Month Value Date to affect the respective month of May.

                                       Click update to commit the changes to the system. 

             

NOTE:   You can only edit an open salary period . Once closed, no modifications can be done.


 

On successful completion you will get a pop up message confirming that the operation was successful.
