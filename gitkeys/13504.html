 Channel Activation Requests.

                          Use this sub - module to receive channel activation requests. 

To access this window, click :Alternate Channels  - > Operations  - >  -> Channel Activation Requests.


 


 

The Following screen will be displayed

Channel Activation Requests fig 1.


 

Importing An Activation Request:


 

click on the import button to import already captured activation requests from an excel sheet saved in .csv format(comma delimited) and the following notification will pop up: 

           

Channel Activation Requests Fig 2.


 

Click on show advanced to view the details of the upload format. See below: 

Channel Activation Requests Fig 3.


 

Select the document containing already captured beneficiaries from an excel sheet saved in .csv format(comma delimited).

See below: 

Channel Activation Requests Fig 4..


 

On successful completion you will get a pop up message confirming that the operation was successful.


 

Trying to upload a file that is currently open will result in the following error : 

Credit Batch Origination Fig 5.

     Amend the entries then upload the correctly captured sheet. 


 

       The following screen will open up :


 

           

Alternate Channels Reconciliation Periods  Fig 6.


 

          The available periods will be displayed.
