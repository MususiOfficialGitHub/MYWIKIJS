Posting Periods

Use this sub-module to set up posting periods. Its the period ie. month, fiscal year etc for which the posting is allowed.

To access this window, click: System Administration - > Setup - > Posting Periods

The following window will open up: 

Creating A Posting Period: 

Click on the Add New button, to open the following screen:

On this Create screen, fill in all the required data in all fields:

Name - Enter the name of the Posting Period e.g. Financial Year 2018

Start Date - Enter the start date of the financial year, e.g. 1/1/2018.

End Date - Enter the end date of the financial year, e.g. 12/31/2018.

NOTE: For posting periods, the start and end dates will always default to the first and last days of the year respectively. 

Type in the new one if the above fields are displayed differently.         

When done click on the CREATE Button to complete this task. 

On successful completion you will get a pop-up message confirming that the operation was successful.

Editing A Posting Period: 

First Identify the posting period to be edited: 

Then click on it and click on Edit against it to open the following window:

On this Edit screen, modify the relevant fields:

Name - Modify the name of the Posting Period e.g. Financial Year 2018

Start Date - Modify the start date of the financial year, e.g. 1/1/2018.

End Date - Modify the end date of the financial year, e.g. 12/31/2018.

Generate Months - Check the flag to generate months for the selected year. The generated Months appear as below:

NOTE: The generated months help in controlling value date postings. 

Identify the month solve restrict and set desired status. 

Is Active - Check the flag to make the period post able. 

NOTE: Be cautious when checking this flag, only one posting period can be active at any given time. 

Activating another one when a current one is in session could interfere with the institution's functioning. 

Is Locked? - check the flag if the period is to be rendered non - usable. 

When done click on the UPDATE Button to complete this task. 

On successful completion you will get a pop-up message confirming that the operation was successful.
