          Employee Exit Approval. 


 

Use this sub-module to approve an exit request for an exiting employee. 

To access this window, click : Human Resources Module -> Navigate -> Operations  -> Exit -> Approve. 


 

 The following screen will open up:


 

Employee Exit Approval Fig 1


 

To Approve An Exit Process: 

           Click on the approval icon 

against the entry, see below:  

   

 NOTE :  You can only approve an exit request whose status is verified. 


 

Employee Exit Approval Fig 2


 

To open the following window:


 

Employee Exit Approval Fig 3

     

     Crosscheck the customer's details to ensure they are correct. 

Approval Remarks - Provide appropriate approval remarks in the approval remarks field then select the accept action from the drop down menu. 


 

              Click on the update button to save the information into the system. 

   

On successful completion you will get a pop up message confirming that the operation was successful.
