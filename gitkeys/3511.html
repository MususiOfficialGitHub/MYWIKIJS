Employee Exit Initiation.

Use this sub-module to initiate an exit request for an existing employee. 

To access this window, click: Human Resources Module -> Navigate -> Operations -> Exit -> Initiate. 


 

 The following screen will open up:        
 

   The already initiated exits are displayed. 


 


 

 Initiating An Exit :


 

 Click on the create button to open the following window:


 

 

Employee Exit Initiation Fig 2


 

       On the above window: 

NOTE :  Employee - This field will be automatically pre-populated  with the data of the user who is currently logged in. 

       The employee's branch, payroll numbers,serial number, identity card number and leave days - if any - will be displayed in the respective fields.


 

 To Capture A New Employee:


 

 Employee -  Click on the look up icon to open the following Employee Look Up dialog box:


 

Employee Exit Initiation Fig 3


 

             Search for and select the customer whose exit process is to be initiated.


 

The employee's Branch, Payroll Numbers, Serial Number, Identity Card Number & Leave Days details will be populated to the respective fields.


 

Type - Select the type of employment termination associated with the employee's exit from the drop down menu options: 

           


 

Reason - Type in the reason(s) for the employee's exiting into this field. 

Document  - Click on the look up icon to open the following windows explorer dialog box: 

           

Employee Exit Initiation Fig 4


 

          Search for and select the document to be uploaded into the system by double clicking on it. 

Title - Type in the title regarding the document being uploaded. 

Description - Type in a brief description about the document into this field.


 

   Assets Tab: 

Employee Exit Initiation Fig 5


 

            All the assets assigned to the employee will be displayed on this tab, together with their total value just below. 

   Always ensure that the employee has no asset alloted to him before initiating an exit. 

  Failure to do so will result in the following message :

           


 

Exit Interview Tab: 


 

Employee Exit Initiation Fig 6


 

          The exit interview questions, both answered and unanswered are  displayed on this tab. 


 

                         Click create to save the information into the system. 

   

On successful completion you will get a pop up message confirming that the operation was successful.
