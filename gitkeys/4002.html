Duty/Workstations

Use this sub module to capture an employer's duty/work stations into the system.

To access this sub module click on: Registry Management =>> Navigate =>> Setup =>> Duty/Work Stations

The following window will open up : 

All the available stations are displayed on the screen

 

Creating A Duty Station:

Click on the add button to open the following screen :

On the above screen :       

Define the parent, type , name and remarks accordingly then create

On successful completion you will get a pop up message confirming that the operation was successful.

Is locked fag-Check is locked to render duty station locked and not operational

 

Editing A Duty Station

Identify the entry to be edited then right click on the entry to see the edit button as shown below:

The Edit Screen will Pop up as below.

View Audit log icon -click to view the specific audit log of the entry selected

Make the changes then Update

On successful completion you will get a pop up message confirming that the operation was successful.
