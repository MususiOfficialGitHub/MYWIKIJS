Loaning Remarks

These are all the reasons  why a loan officer registers, appraises, Disburses or rejects a loan.

Use this sub module to capture all the relevant loaning remarks that will be used during the loaning process.

NOTE: The remarks must me captured to be referenced during the loaning process

To access this module: Credit Management -> Setup -> Loaning Remarks

 The following screen will open up: 

 

Creating Loaning Remarks.    

To  create a loan remark, click on the Add New button to open the following window: 

Name - Type in the loaning remark being captured into this field.

Click on the create button to save the remark into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.

                                  


 

Editing A Loaning Remark.

Identify the Loaning Remark to be edited by right clicking on it.

 Click on the edit Icon against the remark.

The following screen will open up: 

Make the necessary changes to the remark then click on the update button to save the changes into the system.
