Tellers

Use this sub-module to set up the bank tellers in various branches and link them to their respective GL Accounts.

To access this window, click: Front Office Management - > Setup - > Tellers


  Creating A Teller: 

Click on the  add Button, to open the following screen:

On this Create screen :

 On the above screen: 

Name - Enter  the identifying name of the Teller.

Lower Limit - Enter the lower limit of the amount the Teller should have.

Upper Limit - Enter the Upper limit of the amount the Teller can hold at a time.

 [NOTE: If the Teller holds more than the set upper limit, the Teller should Transfer that cash to the Treasury.]

G/L Account - Click on the lookup icon to open a G/L Account Lookup dialog box and Select the appropriate

 G/L: 

Teller Type - Click on the drop down arrow and select the employee type. 

i) Employee Teller Type: 

Shortage G/L Account - Click on the lookup icon to open a G/L Account Lookup dialog box and Select the appropriate G/L Account. 

Excess G/L Account - Click on the lookup icon to open a G/L Account Lookup dialog box and Select the appropriate G/L Account .

Employee - This filed will ONLY be VISIBLE if you selected the Teller Type as Employee.

Click on the search icon and select the correct employee who is designated to be the Teller from the Employee Lookup dialog box:

Use the up down arrows to set the maximum number of transactions that a mini statement can have. 

Is Locked? - Click the flag when the Teller is no longer operational/ being used.

 When done click on the CREATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.

 ii)  POS Terminal (Agent) Teller Type:  

On the above screen: 

Name - Enter  the identifying name of the Teller.

Lower Limit - Enter the lower limit of the amount the Teller should have.

Upper Limit - Enter the Upper limit of the amount the Teller can hold at a time.

POS Agency # - Click on the search icon to open a POS Agency Lookup dialog box and Select the appropriate account: 

Reference - Type in an appropriate reference with regards to the teller being created. 

When done click on the CREATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.

 iii)  ATM Terminal Teller Type:  

On the above screen: 

Name - Enter  the identifying name of the Teller (ATM).

Lower Limit - Enter the lower limit of the amount the Teller should have.

Upper Limit - Enter the Upper limit of the amount the Teller can hold at a time.

G/L Account - Click on the lookup icon to open a G/L Account Lookup dialog box and Select the appropriate G/L: 

Reference - Type in an appropriate reference with regards to the teller being created. 

When done click on the CREATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.

Editing A Teller: 

First Identify the Teller to be edited right click on the entry to be edited:

Then click on the edit icon against it to open the following window:

On this edit screen, make necessary changes to the fields - Name, Lower Limit, Upper Limit, G/L Account, Shortage G/L Account, Excess G/L Account  

Employee - The data on this field can't be modified. 

When done click on the UPDATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.
