Risk Classification Modes

Use this sub module to capture the classification modes for your loans

To access this module: Credit Management -> Setup -> Risk Classification Modes.

The following screen will open up: 

 

 

Creating Risk Classification Mode

Click on the Add New button to open the following window: 

On the above screen: 

Name - define the name of the classification mode.

Negligible Balance - define an amount that will be disregarded as far as classification is concerned. 
Can be left as 0.00

Risk classification details tab:

Days Past Due (Lower Limit) - Define the days due for each classification for example the lower limit for the performing category will be 1, watch 31 etc..

Classification - Select the classification from the drop down list.

Provisioning - Enter the provision percentage per the classification.

Click on the add button after defining a classification and repeat the above process till all the classification(s) are captured 

 

Click the create button.

On successful completion you will get a pop up message confirming that the operation was successful.


 

Editing A Classification Mode.

Identify the Mode to be edited, right click on the mode and select the edit option.

The following screen will open up: 

Make the necessary changes to the mode then click on the update button to save the changes into the system. 

Truncate Decimals? - Select this flag to truncate decimals when checking arrears amounts.

Is Active? - Select this flag to activate the mode to be used to classify loans.

Is Locked? - Select this flag if the mode is no longer being used.

 

Click on the update button to save the changes made on the mode.

On successful completion you will get a pop up message confirming that the operation was successful.
