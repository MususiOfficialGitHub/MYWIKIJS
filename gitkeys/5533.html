Loan Indefinite Charges

Use this sub-module to carry out the recovery of periodic charges for a given loan type. 

    e.g insurance can be recovered periodically over the term of the loan.

To access this window, click :Credit Management - > Operations - >Recurring Procedures - > Loan Indefinite Charges


 

The following screen opens up :

Loan Indefinite Charges Fig 1.

All the loans products from which the charge should be recovered are displayed on the screen. 


 

 Loan Indefinite Charges Recovery :

 To recover indefinite charges on loan products : 

Posting Period - The active posting period normally populates to this field. To select an alternative one, click on the lookup icon and pick the appropriate. 

Month - Select the appropriate month for which the charges are to be recovered from the drop down menu : 

Loan Indefinite Charges Fig 2.

Priority - Select the relevant recovery priority from the drop down menu : 

Loan Indefinite Charges Fig 3.

Execution  Filter - Select the appropriate capitalization filter from the drop down menu : 


 


 

Check this flag if the value date is to be enforced on the entries during the charge recovery process. 


 

Select the loan product's cost center from the drop down menu, either FOSA or BOSA. 

Refresh to reload the loan products  on to the screen. 

Select the products from which the charge will be recovered by selecting the check box against the respective loan product : 

Loan Indefinite Charges Fig 4. 

Use the toggle selection flag  

 if all the available loans are to be affected by the charge recovery. 

Click on the update button to effect the charge recovery. 


 

   

On successful completion you will get a pop up message confirming that the operation was successful.
