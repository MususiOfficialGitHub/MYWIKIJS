Credit Batch Authorization

     Use this sub-module to authorize the verified Credit Batches. 

To access this window, click :Accounts Management - > Navigate - > Operations - >Batch Authorization ->  Credit


 

The following screen will open up:

Credit Batch Authorization Fig 1.


 

   The verified credit  batches will be displayed on the screen. 

REMEMBER: Select a Batch Type before clicking the refresh button to get all batches that were created for that selected type then use Start and End date to search for the specific batch to be authorized. For example if Payout is picked from the above, only Payout batch types will be viewed on clicking the refresh button.

NOTE: You can only verify a batch whose status is verified. Check the status column. 


 


 

    Credit Batch  Authorization :


 

           Identify the credit batch to be authorized :

           Click on the verification icon 

against the entry to open the following screen : 

 

Credit Batch Authorization Fig 2.

Cross check and ensure the entry details are correct. 

Remarks - Provide authorization remarks in the remarks field. 

        Pick the post option from the drop down menu.

        Click on the update Button to effect the verification. 

On successful completion you will get a pop up message confirming that the operation was successful.
