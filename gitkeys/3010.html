Performance Pillars

 Use this sub-module to create Performance pillars into the system.

To access this window, click: Human Resources Module -> Navigate -> Setup - > Performance pillars.

The following screen will open up:

 

Creating Performance pillar :

Click on the Add new button to open the following screen:

On the above screen: 

Provide the Name & Weight in the respective fields.

Click Create button to add the period into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.

 

Editing   Performance Pillars:

Identify the performance pillar record to edit and right-click on it.

Click on the edit icon to open the following screen:

On the above screen:

Make changes to the Name & Weight fields. 

if Pillar is no longer in use, check the is Locked? as shown below.

 Click Update to commit the changes into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.
