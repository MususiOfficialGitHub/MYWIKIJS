Catalog

In this sub-module, all the movements involving any files that have been dispatched and received back into the registry will be listed here. 

To access this window, click :Registry Management =>> Navigate =>> Operations =>> File Tracking =>> Catalog

The following window will open up: 

Viewing Of File Movement History.

To view more details about a specific file right click on the file to see the view icon as shown

Click on view file movement to view the file movement history

Click on view audit log to see the log history of the file

Click on the view icon to view the details
