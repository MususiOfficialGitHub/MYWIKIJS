 Data Periods

    This module is used  for creating (opening) a Data Period, for example a month in a Financial Year.

To access this window, click :Credit Management - > Operations - > Data Processing - > Data Period


 

The following screen will open up:


 

Data Period Fig 1


 

       Creating A Data Period

To create a data period,click on the CREATE Button to open the following window:

Data Period Fig 2

      On the above scree:

Input the following information:

Posting Period - The system normally defaults to the active posting period. 

                        To change this, click on the Lookup Icon to open the "Posting Period Lookup" Dialog Box and select the appropriate posting period. 

Credit Type -Look up for the appropriate credit type 

Month - Select the appropriate month from the drop down menu. 

Remarks -  Provide relevant remarks concerning the data period into this field.

                  Click on the create button to save the period into the system.


 

 

On successful completion you will get a pop up message confirming that the operation was successful.


 

                                 


 

     To confirm the created Data Period, click on the Data Periods and then click on the REFRESH Button.


 

Editing A Data Period: 

     To  edit  a data period: 

Identify the period to be edited; 

             Data Period Fig 3

          Then click on the edit icon 

against it to open the following window:

Data Period Fig 4 


 

               Make the necessary changes to the data period then click on the update button to save the changes into the system. 


 

 

On successful completion you will get a pop up message confirming that the operation was successful.

                                 

NOTE : Once a data period for a certain month is closed, it cannot be amended. 

                          Once a data period for a particular month is created - whether its status is closed or open, no new data period can be created within the same month.
