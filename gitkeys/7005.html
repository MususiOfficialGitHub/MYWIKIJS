Package Types

 Use the package type module to create different package types applicable for transportation and storage of goods/products.

To access this sub module click on Procurement and Assets Management-> Navigate -> Setup -> Package Types 

The following screen will open up : 

The available package types are displayed on the screen. 

Creating A  Package Type : 

Click on the ADD NEW button to open up the following window : 

 Provide data in the fields. 

Name - Enter the name of the package type.

Remarks - Type in a brief remark regarding the package type, giving more insight to the package type.

Click on the update button to save the package type into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.

 

Editing  A Package Type : 

Identify the package type to be edited :

Click on the edit icon against the entry to open the following window : 

Make applicable changes to the name & remarks fields.

Click on the update button to save the changes into the system .

On successful completion you will get a pop up message confirming that the operation was successful.
