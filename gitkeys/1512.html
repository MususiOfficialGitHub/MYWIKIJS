The Vanguard Financials Audit Trail is a very key component, with particular regards to access of the system. 

The system automatically keeps track of actions; login, logout, roles addition, password changes and user creation - for every user.

Captured entities include : Time Logged in, Application User & Environment User Name, operating system, Environment Domain Name & Process Id are among other details. 

To access this module click on System Administration  --> Navigate --> Operations --> Risk & Audit --> Audit Trails
