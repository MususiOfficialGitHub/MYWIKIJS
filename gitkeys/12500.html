Groups

Use this module to create and manage micro credit groups.

To access this window, click :Micro-credit Module -> Navigate -> Operations ->Groups

The following screen will open:

How To Create Micro Credit Groups 

1. To create a Micro - credit group, click on the ADD NEW button to open the following screen: 

2. Name - Click on the lookup icon to open the following Customer Look Up dialog box: 

From the above list, select the customer to be attached to the micro credit group. 

3. Credit Officer- Click on the lookup icon to open the following Micro Credit Officer Look Up dialog box: 

From the above list, select the micro credit officer to be attached to the micro credit group. 

4. Parent Group - Click on the lookup icon to open the following Micro Credit Group Look Up dialog box:  

5. Type- Click on the drop down menu to Select the type of micro-credit group you're creating.

6. Purpose- Type in a brief note on the core purpose of the micro-credit group being created.

7. Activities- Type in description of the activities the micro-credit group will be engaging in.

8. Meeting Frequency- Click on the drop down menu to Select  the frequency at which the micro credit group will be meeting. i.e weekly,monthly,semi-annual e.t.c

9. Meeting Day Of The Week - Select from the list the day of the week the micro-credit group will be holding meetings. 

 Click on the drop down menu  then select the appropriate day. 

10. Meeting Place - Type in the place where the micro-credit group meetings will be taking place. 

11. Minimum Members- Using the arrows to decrease and increase the number of members by 1, select a number of minimum number of members allowed in the group. The upward  arrow increases the number by 1 while the downward farrow decreases the number by 1.

12. Maximum members -Using the arrows to decrease and increase the number of members by 1, select a number of maximum members allowed in the group. The upward  arrow increases the number by 1 while the downward  arrow decreases the number by 1. 

13. Remarks - Type in relevant remarks concerning the group.

14. Click on the create button to finish creating the micro-credit group.

 Adding Members To A Micro-Credit Group: 

On the member details pane, fill in the following: 

Customer - click on the customer lookup icon to open the following  Customer Lookup dialog box: 

Employer -  data will be automatically populated to this field when the customer is selected. 

Station - data will be automatically populated to this field when the customer is selected. 

Account Number - data will be automatically populated to this field when the customer is selected. 

Membership Number - data will be automatically populated to this field when the customer is selected. 

Personal File Number - data will be automatically populated to this field when the customer is selected. 

Designation -  select a customer's designation by clicking on the drop down arrow to reveal  the list of designations.

From the above list, select the correct designation to which the customer will be attached. 

Loan Cycle - Select the loan cycle for the member by using the up/ down scroll keys. 

Remarks - Type in a brief remark about the member in this field.

Once done filling the above details, click on the add button to  append the data to the group members pane, see below: 

To add more members repeat the above procedure. 

Removing A Member From A Micro-Credit Group: 

Identify the member to be removed from a particular micro credit group. 

Select the member . 

Once selected, click on the remove button to effect the process. 

Click on the create button to save the details into the system.

On successful completion you will get a pop up message confirming that the operation was successful.

How To Edit Micro Credit Group(s) 

To edit a micro credit group, click on the edit icon against the group to be edited

The following screen will open up: 

On the above screen, edit the following : 

Name - the data on this field cannot be modified.

Credit Officer- Click on the lookup icon to open the following Micro Credit Officer Look Up dialog box: 

From the above list, select the new micro credit officer to be attached to the micro credit group. 

4. Parent Group - Click on the lookup icon to open the following Micro Credit Group Look Up dialog box:  

From the above list, select the new micro credit parent group to be attached to the micro credit group. 

5. Type- Modify the data on this field by clicking on the drop down menu to Select the new type of micro-credit group. 

 

6. Purpose - modify core purpose of the micro-credit group.

7. Activities - Modify the activities the micro-credit group will be engaging in.

8. Meeting Frequency - Click on the drop down menu to Select  the new frequency at which the micro credit group will be meeting. i.e weekly,monthly,semi-annual e.t.c

9. Meeting Day Of The Week - Select from the list the day of the week the micro-credit group will be holding meetings. 

   Click on the drop down menu  then select the appropriate day. 

10. Meeting Place - Type in the new place where the micro-credit group meetings will be taking place. 

11. Minimum Members- Using the arrows to decrease and increase the number of members by 1, select a number of minimum number of members allowed in the group. The upward  arrow increases the number by 1 while the downward farrow decreases the number by 1.

12. Maximum members -Using the arrows to decrease and increase the number of members by 1, select a number of maximum members allowed in the group. The upward  arrow increases the number by 1 while the downward  arrow decreases the number by 1. 

13. Remarks - Type in relevant remarks concerning the group.
