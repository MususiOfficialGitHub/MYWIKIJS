Roles

Roles in Vanguard Financial systems determine what a user does.

The user permissions are determined by various system roles. 

Each role can be fine-tuned by adding permissions, which grant the user access to specific modules & operations. 

Click on System Administration from the menu items; System Administration -->Operations--> Roles.

The following screen opens up: 

The already captured roles are displayed. 

 

Creating A New Role

Click create to open the following window: 

On the above screen: 

Role Name - Type in the name of the role being created. 

Click create to save the role into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.

Refresh to view the newly created roles.


Viewing A Role

Identify the role to be viewed:

Click on the view icon against the role to open the following screen: 

Users In Role:

Users defined in the role are displayed under this tab. 

Modules ACL:

All the modules that the users in the role can access are displayed under this tab.

Operations ACL:

All the operations that the users in the role can carry out are displayed under this tab.
