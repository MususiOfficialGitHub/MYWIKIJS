Funeral Rider Claims

Use this sub-module to capture funeral rider claim payables applications into the system. 

To access this module click on 

 Registry Management =>> Navigate =>> Operations =>> Membership Withdrawal =>> Settlement =>> Funeral Rider Claim

The following screen will open up:

Creating A Funeral Claim Application:

To capture a new funeral rider claim application, click on the add button to the bottom right of the screen to open the following screen: 

On the above screen, do the following: 

Claim Type - Click on the drop down to select the appropriate claim type being originated, member claim or spouse claim. 

On this screen, click on the customer search icon to open the following customer look up dialog box: 

Select the customer whose claim is to be processed from the above list.. 

If at the claim type you chose spouse claim then the spouse claim then you will see a spouse details area for you to fill. 

All fields will populate automatically after selecting the customer from the lookup.

Claimant Details Pane:

Fill in the Details under the Signatories details area as well as the claimant details/spouse details

 On this screen, fill in the following: 

Claimant Names - type in the claimant's names into this field

Claimant Mobile Number - type in the claimant's mobile number into this field. 

Claimant Id Number - type in the claimant's Id number into this field. 

Relationship - type in the relationship of the claimant to the deceased

Signature Date - select the date when the signature for claiming was appended. 

Signatories Details Pane:

 Signatories are just typical witnesses for the application of the funeral rider payable. 

On this screen, fill in the following: 

Chief Names - type in the name of the member's chief into this field.  

Chief Id Number - type in the identity card number of the member's chief into this field.

Immediate Superior Names - type in the name of the immediate superior to the member's chief into this field.

Immediate Superior Id Number -  type in the identity card number of the immediate superior to the member's chief into this field.

Delegate Names -  type in the name of the member's delegate into this field. 

Delegate Personal file Number - type in the personal file number  of the member's delegate into this field. 

Board Member Names - type in the name of the designated board member into this field. 

Board Member Personal file Number - type in the personal file number of the designated board member into this field. 

Signature Date - select the date when the signature for claiming was appended.

Once done with the above steps, click on the create button to save the information into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.

Editing Funeral Rider Claims:

Identify the entry to be modified then right  click on the entry for the edit icon 

To open the following screen:

On the above screen, modify the spouse details and signatories. Member details cannot be modified. 

You can also modify the death date, title, document, remarks and description. 

Click on the update button to save the changes into the system.

On successful completion you will get a pop up message confirming that the operation was successful.
