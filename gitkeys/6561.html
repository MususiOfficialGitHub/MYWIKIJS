Hierarchies.

    Use this sub-module to set up the institution's  Budget.

To access this window, click :Accounts Management - > Operations- > Budget Management - > Hierarchies. 


 

The following screen will open up :

Budget Hierarchy Fig 1.


 

     Creating A Budget Period : 

            Click on the CREATE Button, to open the following screen:

Budget Hierarchy Fig 2.

     On the above window :

Name - Enter the name/ description of the item being created

Parent hierarchy -  Lookup  select the appropriate parent if non create one under header entry

Category-Pick up the header(Non postable) or detail(postable) depending on the entry being created

Gl Account- lookup the appropriate gl account on the following window 


 

On successful completion you will get a pop up message confirming that the operation was successful.


 

                                     


 


 

 Editing  A Budget Period :

          Identify the Budget period to be edited.

    Click on the edit icon 

against it to open the following window : 

Budget HierarchyFig 3.

Make necessary changes to the affected fields then click create

    

On successful completion you will get a pop up message confirming that the operation was successful.
