Unit Of Measure

Use this sub module to create units of measurement for your inventory and asset as well. 

You may create custom units of measurement or standard units of measurement which you will later use to quantify assets and inventory.

To access this sub module click on: Procurement and Assets Management -> Navigate -> Setup -> Unit of Measure

The following window will open up : 

Creating A Unit of Measure : 

Click on the Add button to open the following window : 

Fill in the form with data in the relevant fields.

Name - Enter the name of the unit of measure.

Contains  - Select the number of units the measurement contains this will co-respond to the "Of Base Units" using the up down scroll keys.

Of Base Units - Select the base unit to be associated with the unit if measure from the drop down menu.

Click on the Create button to save the entry into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.

             
  Editing A Unit Of Measure :

Identify the unit of measure to be edited : 

Click on the edit icon to open the following window : 

Make the necessary changes to the Name, Contains & Of Base Units fields. 

Click on the update button to save the changes into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
