Documents

    Use this sub-module to add Documents that relate to particular agencies Registered

To access this window, click ::Alternate Channels  - > Operations  - > Agency Register =>> Documents


 

Click on the documents and the Window below will appear. 

Agency Documents Fig 1

Creating a Record for Documents

Proceed to click Create to make a new entry.

Agency Documents Fig 2


 

In the above page fill in the details. Use the POS agent lookup to select the POS Agent like shown below.

Agency Documents Fig 3


 

Proceed to fill in the rest of the details as shown below

Agency Documents Fig 4

When Done click create.

On successful completion you will get a pop up message confirming that the operation was successful.

                       


 

Editing Documents

To edit a document, on the default documents page. Click on the edit icon 

against the record you intend to make changes.

That opens the page as shown below

Agency Documents Fig 4

The details of the records appear as shown.  The POS agency field is normally disabled for editing thus you can only change the rest of the fields.

After cross-checking and making changes. Click the update button 

On successful completion you will get a pop up message confirming that the operation was successful.


 


 

Viewing Documents

Click on the edit icon 

against the record that you intend to view. 

On clicking a save window prompt appears as shown below. 

 

Agency Documents Fig 5

The windows pop up prompts you to save the file that you have intended to view. it saves as pdf by default. choose the location you intend to save the file and Click on Save. 

Saving gives a confirmation pop up on vanguard as below.

On successful completion you will get a pop up message confirming that the operation was successful.


 


 

Proceed to open the file from where you saved it from.
