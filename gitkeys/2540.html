Signatories

Use this sub-module to add signatories to a customer's account.

To access this window, click: Front Office Management - > Operations - > Customers Account - > Signatories.

The following screen will open up:

 

Capturing Account Signatories

Identify the account whose signatories are to be captured: 

Right-Click on the member to show the edit icon against them to open the following screen: 

The customer's details will be displayed on the upper part of the screen: 

Signatory Details:

Fill all the signatory's details. 

Salutation - select the signatory's salutation from the drop-down menu.

Gender - Select the correct gender. 

Relationship - Select the signatory's relationship with the account holder from the drop-down menu.

Identity Card Type - Select the signatory's Identity Card Type from the drop-down menu.

Provide the signatory's First Name, Other Names, Identity Card Number, Address Lines 1 & 2, City, Postal Code, Physical Address, Land Line, E-mail. 

Mobile Line - Enter his/her cell phone number: 
NOTE: start with plus (+), followed by country code i.e. +254712345678

Remarks - Provide applicable remarks. 

 

Click on CREATE to commit the signatory to the system & signatories' pane just below: 

Repeat the above steps to add more signatories to the selected account. 

In case of an erroneous entry, check the select box against the signatory, then click on the remove button to effect the removal.  

On clicking the remove button, you will get the following confirmation message:

 

NOTE: This operation does not involve a system acknowledgment. 
By virtue of clicking the create button, the signatories are captured into the system. 

 

View Account Signatories

To view the captured signatories, find the target account then right-click on it:

 Click on view to display the signatories:
