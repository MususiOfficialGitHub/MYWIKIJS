External Agencies

Use this sub-module to set up external agencies and link them to a GL account and Branch.

To access this window, click: Front Office Management - > Setup - > External Agencies.

The following screen will open up.

Creating an External Agency: 

Click on the Add New Button, to open the following screen:

Agency Type - Click on the Agency Type drop down to select an Agency Type.

Name - Enter the name  of the Treasury (probably by branch name)

Chart Of Account - Click on the lookup icon to select the appropriate G/L Account from the G/L Account Lookup dialog box: 

Branch - Click on the lookup icon  to select the associated Branch name of the treasury being created from the Branch Lookup dialog box:

When done click on the CREATE Button to complete this task.

On successful completion you will get a pop up message confirming that the operation was successful.

Editing an Eternal Agency:

First identify the External Agency to be edited: 

Right click on it then click on  the edit icon against it to open the following screen: 

On the above screen, make the necessary changes to the Agency Type, Name and G/L Account.

 When done click on the UPDATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.
