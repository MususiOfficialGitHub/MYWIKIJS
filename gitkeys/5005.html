Loan Repayment Modes

 Use this sub module to capture the repayment modes that will be used during loan processing

To access this module: Credit Management -> Setup -> Loan Repayment Modes

The following screen will open up: 

 

Creating Loan Repayment Mode 

To  create a loan repayment mode, click on the Add New button to open the following window: 

On the above screen: 

Name - Type the name of the repayment mode.

Standing Order Trigger - Select a standing order trigger that will be associated with the repayment mode.
When the loan is verified/disbursed, the standing order created will have the trigger of the repayment mode, which may differ from what is set on the loan product setup.

Credit Type - Select a credit type that will be associated with the repayment mode.
Standing orders will only execute when the selected credit type is used in Credit Batch Procedures. 

Is Locked? - Select this flag if the repayment mode is not to be used.

 

Click the create button once done.

On successful completion you will get a pop up message confirming that the operation was successful.

 

 

Editing A Loan Repayment Mode.

Identify the Loan Repayment Mode to be edited. 

Right click on the mode and select the edit option against the mode.

The following screen will open up: 

Make the necessary changes to the mode then click on the update button to save the changes into the system. 

Default Mapping Details Tab: 

Use this tab to capture the employers whose members will have a default mapping for loans to be repaid using the repayment mode.

Business/Group/Employer - Select an employer to set their default mapping for the mode.

Loan Product - Select a loan product to be repaid using the repayment mode for the specific employer/business/group selected.

In case of an erroneous entry, you may right click on the entry and select the remove option to remove it from the default mapping settings.

Is Locked? - Select this flag if the mode is no longer in use.

 

On successful completion you will get a pop up message confirming that the operation was successful.
