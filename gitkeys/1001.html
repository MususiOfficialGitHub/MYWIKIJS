Currencies

Use this sub module to capture currency.

Click on System Administration from the menu items; System Administration -->Setup--> Currency.

The following screen opens up as the available currencies are displayed: 

 

Creating a Currency

Click the Add New icon to open the following window:

On the above screen:

Name - Type in the name of the currency.

Symbol - Enter currency symbol.

ISO Code - Type in the ISO Code of the currency.

ISO Number - Define ISO number.
 

Once done click on create and the following screen should pop up once successful. 

 

Editing a Currency

In order to edit an existing currency, right-click on the item you want to edit, then click on the edit icon that appears as shown below:

The following screen will pop up:

On the above screen, edit the necessary fields accordingly.

Is Locked? - This is a flag that renders the currency unusable when ticked.

 

Once you're done editing, simply click on update and the following window will pop up if successful:
