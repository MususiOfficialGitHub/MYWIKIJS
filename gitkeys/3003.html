Holidays

Use this sub-module to setup public holidays.

Creating public holidays in the HR Module of Vanguard Financial s is important as it helps in counting of an employee's leave days among other purposes.

To access this window, click: Human Resources Module -> Navigate -> Setup - > Holidays.

The following screen will open up:

All captured holidays are displayed as shown.

Creating A Holiday:

Click on the add new button to open the following screen: 

 

Select the holiday's Posting Period from the Posting Period Look Up dialog box:

 Input the holiday's Name, Start Date & End Date.


Click on create button to create the holiday into the system.

 All captured holidays are displayed.

On successful completion you will get a pop-up message confirming that the operation was successful.

 Refresh to view the created holiday.


 

Editing A Holiday:

Identify the holiday to be edited and right-click to display as shown window.
 

Click Edit to open the as shown window.

 Check the is locked? if the holiday is to be rendered useless/ inactive. 


 Click update button to save the changes into the system or Cancel to do away with changes 

On successful completion you will get a pop-up message confirming that the operation was successful and press okay.
