Text Alerts

Use this module to create and view all the Text Alerts that have been received or sent out of the system.

The origin of the messages (that's internal or external) as well as the recipient of the messages are also displayed.

This sub module is also used to send messages in bulk to specific departments / groups within the SACCO (internally).

To access this window, click on; Customer Relationship Management -> Operations -> Messaging- > Text Alert.

The text alerts have been categorized into 6 sub-groups. Select the category you would like to view as shown in the figure below and then click on the refresh button.


 

Creating a Text Alert

 To create a Text Alert, click on the “Add New” button to open up the screen below: 

On the screen above:

Message - Enter the intended message that's to be sent to the specified group. 

Recipients - Choose by ticking the checkboxes next to the appropriate recipient group.

NOTE: The recipient groups being selected here were previously captured in the Message Groups set-up section. 

To set up a new message target group, please refer to Customer Relationship Management- > Setup - > Messaging Groups.

Append Signature - Choose by ticking the checkboxes next to the append signature to append the company's signature in the message. 

To send the message to the selected target group, then click on the CREATE Button.

Automatically the message will be sent to the selected group.

On successful completion you will get a pop-up message confirming that the operation was successful.

 

Creating a Quick Text

Click the Quick Text icon to open the following window:

Message - Enter the intended message that's to be sent. 

Recipient - On the Customer name select a customer to auto fill the recipient's details as shown below.

Click the add icon to add the Customers to the text alert.

Target Customers - on the target customer select the intended individual customers.

Mismatched Import Entries - Mismatched entries will appear here if when importing a list of recipients to send text alerts to their Payroll Number doesn't match or is missing in the system, which means the system can't find them, and those texts alerts won't be sent. Please make sure all the payroll numbers are correct to ensure everyone gets the message.

Append Signature - Choose by ticking the checkboxes next to the append signature to append the company's signature in the message. 

Click Add button to create a text alert.

On successful completion you will get a pop-up message confirming that the operation was successful.
