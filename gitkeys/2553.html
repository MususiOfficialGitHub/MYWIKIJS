Approval

Use this module to approve a registered account closure request.

To Access this sub module, click On Front Office Management->Operations ->Account Closure ->Approval.

The following screen opens up:

On the above window:

Full Account Number - Click on the look up icon to open the Customer Account Closure Request Lookup dialog box:

Select the status of the request to be approved as registered. 

Select the correct date range.

Narrow down the filter using the search strings available, then select the entry to be verified. 

Data will be populated to the relevant fields.

Cross check through the other tabs to view more details about the customer.

Approval Remarks - Enter the approval remarks appropriately.

 

Select the approve action from the drop-down menu then click on the update button effect the approval. 

On successful completion you will get a pop-up message confirming that the operation was successful.
