Charges Exemptions

Use this module to exempt customers from certain charges.

To access this sub module Click on. 

The following window will open up:

Creating A Charge Exemption

Click on the Add New button to create a new charge exemption to open the following screen:

On the above window, fill in the following:

Name - Enter the name of the charge exemption being created. 

Charge - Click on the charge lookup icon to open the following Charge Lookup dialog box :

 Select the charge from which you want to exempt customers.

Exempt Customer Details Pane:

 Add the customers you want to exempt from the charge:

 Click on the customer look up icon to open the following  Customer Lookup dialog box: 

 Search for and select the customer to be added to the Exempt Customer Details pane. 

Employer -  Data will be automatically populated to this field when the customer is selected. 

Station -  Data will be automatically populated to this field when the customer is selected. 

Account Number  -  Data will be automatically populated to this field when the customer is selected. 

Identification Number - Data will be automatically populated to this field when the customer is selected. 

Domicile Branch - Data will be automatically populated to this field when the customer is selected. 

Duty Work station - Data will be automatically populated to this field when the customer is selected. 

Personal File Number -  Data will be automatically populated to this field when the customer is selected. 

Membership status -  Data will be automatically populated/ made inactive for any data entry to this field when the customer is selected. 

Remarks -  Type in appropriate remarks into this field.

Click on the add button 

to add the customer to the Exempt Customers pane just below: 

NOTE : The above steps have to be carried out for every customer that is to be added to the Exempt Customers pane/  exempted from the charge being defined. 

Along the same lines, a customer who was erroneously captured or for some other reason added to the exemption list wrongly can be removed from the list. 

This is done by selecting the customer in the Exempt CustomerPane, then clicking on the Remove Button: 

Click on the create button to save the information into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.

Editing A Charge Exemption:

Charge exemption editing occurs when there's the need to modify the charges details - name or customers. 

Identify the charge to be edited then click on the edit icon 

against it to open the following screen: 

On the above window, make modifications appropriately. 

NOTE : Editing in this case is just limited to modification of the charge name and addition or removal of customers from the exemption list. 

To add more customers to the exemption list: 

Click on the customer look up icon to open the following  Customer Lookup dialog box: 

Search for and select the customer to be added to the Exempt Customer Details pane. 

Employer -  Data will be automatically populated to this field when the customer is selected. 

Station -  Data will be automatically populated to this field when the customer is selected. 

Account Number  -  Data will be automatically populated to this field when the customer is selected. 

Membership Number -  Data will be automatically populated to this field when the customer is selected. 

Personal File Number -  Data will be automatically populated to this field when the customer is selected. 

Remarks -  Type in appropriate remarks into this field.

Click on the add button 

Viewing Customers Exempted From A Charge

Click on the view icon

against the charge type you want to view.  All the customers exempted from the selected charge will be displayed on the Exempt Customers separate window
