Concession Exceptions

Use this sub-module to capture specific members' concession exceptions.

Concession is the amount of money to be retained in a member's account after all deductions have been made.

The concession amount/ percentage is defined on the batch and is uniformly applied to all  accounts that will be affected.

Any member whose concession amount/ percentage is variant from the one defined on the batch should be captured here.

NOTE :  Concession exceptions are only applicable to credit types

To access this sub module Click on:

Registry Management =>> Navigate =>> Operations =>> Members Management =>> Concession Exceptions

The following window opens up: 

Creating A Concession Exception

On the above window click add button.

The following window opens up :

1. Define an appropriate name for the exception.

2.Select the credit type to be affected by the exceptions  from the search icon.

3. In the Excepted Customer Details pane  ;

         a) select a member with a concession exception

         b) select the member's concession type and define the corresponding concession and remarks then add.

The above step should be done for each and every member with an exception then added to the list of Excepted customers just below. 

Use the remove button by right clicking on the entry selected to get rid of any erroneous entries as shown below

Click create once done.

On successful completion you will get a pop up message confirming that the operation was successful.

 

Editing A Concession Exception : 

Identify the credit type to be modified right click on the entry :

Click on the edit button to open the following window :

Make changes to the exceptions entries ~ add or remove any entries

Update once done.
