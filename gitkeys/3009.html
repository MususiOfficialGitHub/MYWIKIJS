Training Types

Use this sub-module to capture training types.

To access this window, click : Human Resources Module -> Navigate -> Setup - > Training Types

The following window will open up :

Available entries are displayed. 

 

Creating A New Training Type :

Click on the Add new button and the following screen will open up :        

Name - Define the training type.

Click on the create button. On successful completion you will get a pop up message confirming that the operation was successful.
