Departments

Use this sub-module to setup new departments.

To access this window, click: Human Resources Module -> Navigate -> Setup - > Departments.

The following window will open up:

All the available departments are displayed. 

 

Creating A New Department

 Click on the create button and the following screen will open up :        

Description: Click on the add new button to add a new department, it will take to the window below

Name - Enter the name of the new department in the space provided.

Click on the create button. 

Is Locked? - Click the flag when the department is no longer operational/ being used.

On successful completion you will get a pop up message confirming that the operation was successful as shown below and press okay.                       

Refresh to see the new department.

Editing A Department 

Identify the department to be modified. 

Then click on the edit icon to open the next window

Make applicable changes to the department name. 

Check if it's to be a registry department.

Check if it's to be put out of operations. 

Update to save changes.

On successful completion you will get a pop-up message confirming that the operation was completed successful.
