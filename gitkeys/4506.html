Next-of-Kin

Use this module to register, view and edit a customer's next of kin.

The following window will open up:

The next of kin register is displayed according to the default grouping.

NOTE : To capture a customer's next of kin, we make use of the edit function in this case: 

Identify the customer whose next of kin(s) are to be captured. 

Make use of the grouping and search options to find the customer's account. 

Right click on it and select Edit icon

The screen below will pop up :

On the above screen, fill in the details as follows: 

  • Salutation - Select the kin's salutation from the drop down menu
  • Gender - Pick the right gender
  • Relationship - Pick the kin's relationship to the customer
  • First Name - Type in kin's first name
  • Other Names - Type in kin's other names
  • Education level - Choose the education level for the kin
  • Nominated % -  select the nomination percentage for the kin.
  • Id Card Type - select the kin's identity card type from the drop down menu
  • Id Card Number - type in the kin's id card number
  • Address Line 1 - Provide the kin's primary address
  • Address Line 2 - Provide the kin's secondary address if any.
  • City - type in the kin's current city
  • Postal Code - provide the city's postal code
  • Street - Type in the street name on which the kin is located
  • Mobile Line - type in the kin's mobile number, take into consideration the convention used.
  • Email - Provide the kin's email address
  • Land-line - Provide the kin's land line number if any.
  • Remarks - Type in appropriate remarks into this field.

Fill in the above details into the respective fields as shown below: 

NOTE : For nominees who are under 18years i.e. without ID, enter N/A or any other meaningful description.

The total nominated percentage should not exceed 100%. 

Click on the add button to append the added nominee to the Next-Of-Kin Preferences Pane:

NOTE : For every new kin that is to be captured, the former steps are to be carried out on each and every one of them then added to the Next - Of - Kin Preferences pane until they're all captured. 

 Along the same lines, an erroneous or unwanted entry can also be removed from the preferences pane. 

The remove button is used to get rid of an erroneous entry from the kin preferences pane by right clicking on  the entry then clicking on the remove button: 

 Click on the update button to save the information into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
