Cost distribution rules

To set up a cost distribution rule  or view existing cost distribution rules  follow the illustration below;

From the Accounts Management -> Setup -> G/L Accounts - > Cost Distribution rules.  

The following screen will open up: 

Creating A Cost distribution rule

To add a new cost distribution rule, Click on the ADD NEW  button to open the following screen: 

On the above screen, provide an appropriate name in the text box provided 

Distribute the cost on the cost center pane.Look up the cost center provide  a name and the percentage.

Click on the add button to add more entries.

Click on the Create button to save the new cost distribution rule into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.

NOTE: To view a newly added cost distribution rule , click on the Refresh button. The list will be updated to display the new cost distribution rule.

Update or Create accordingly, depending on the context of the operation being carried out. 

Editing A Cost Distribution rule

To modify a cost distribution rule, first identify the cost center: Then click on the edit icon against it to open the following screen:

Modify the cost distribution rule accordingly then  click on the update button to save the changes into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
