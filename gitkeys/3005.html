Leave Types

Use this sub-module to create different leave types into the system. 

To access this window, click: Human Resources Module -> Navigate -> Setup - > Leave Types. 

The following screen will open up:        

All the available leave types are displayed. 

Creating A Leave Type 

Click on the Add new to open the following window:

On the above window: 

Define the leave type's Name, Days Category, & Maximum Days details.

  Check 

if the days reduce once the leave type is approved.

Click Create to create the leave type into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.

 

Editing A Leave Type 

 Identify the leave type to be edited and right-click.

Click on the edit icon to open the following screen:

Make changes to the Name, Days Category & Maximum Days fields. 

Click the Update button to apply the changes into the system.  

On successful completion you will get a pop-up message confirming that the operation was successful.
