Officers

Use this module to create and manage micro-credit officers. 

To access this window, click :Micro-credit Module -> Navigate -> Setup ->Officers

The following screen will open:

How To Create Micro-Credit Officers

1. Click on the ADD NEW  button to open the following window: 

Employee Name - Search for the employee from the customer register by clicking on the employee lookup icon. 

The following Employee Look Up dialog box will be opened:

Note: Someone must exist in the employee register in order to be created as an Officer. 

Remarks- Enter relevant remarks concerning this micro-credit officer into this field.

Check if you want to lock a certain officer from micro-credit 

Click on the CREATE button to create the officer.

On successful completion you will get a pop up message confirming that the operation was successful.

How To Edit A Micro-Credit Officer

To edit a micro credit officer, identify the officer whom is to be edited. 

Click on the edit icon against the officer to open the following screen. 

The following screen will open:
 

2. Employee Details -  The data in this fields cannot be edited. 

3. Remarks - Modify  comments or remarks. 

4. When done click on UPDATE button.

On successful completion you will get a pop up message confirming that the operation was successful
