Survey Groups

Use this sub module to setup Survey groups.

To access this window, click: Customer Relationship management - > Setup - > Survey groups.

The window below opens up: 

 

Creating A Survey Group

Click the Add New icon to open the following window.

Name - Enter the name of the message group. 

Target - Pick an appropriate target group from the drop-down menu.

On the membership status you can select from the drop down below the status choice.

 

Click create to commit the changes into the system.

On successful completion you will get a pop-up message confirming that the operation was successful.


 

Editing Survey groups

In order to edit an existing survey group, right-click on the item you want to edit, then click on the edit icon that appears as shown below:

 

Make modifications then click update to commit the changes into the system.

On successful completion you will get a pop-up message confirming that the operation was successful.
