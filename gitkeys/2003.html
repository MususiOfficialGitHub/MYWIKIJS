Petty Cashiers

Use this sub-module to set up the Petty Cashiers in various branches and link them to their respective GL Accounts.

To access this window, click: Front Office Management - > Setup - > Petty Cashiers

The following screen will open up: 

Creating A Petty Cashier: 

Click on the Add New Button, to open the following screen:

Name - Enter the name of the Petty Cashier. (Probably by branch name)

G/L Account - Click on the lookup icon to select the appropriate G/L Account from the G/L Account Lookup dialog box: 

Branch - Click on the lookup icon to select the associated Branch name of the treasury being created from the Branch Lookup dialog box:

When done click on the CREATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.

 Editing A Petty Cashier: 

 First identify the Petty Cashier to be edited: 

Then click on the edit icon against it to open the following window:

Name - Modify the name of the Treasury (probably by branch name)

G/L Account - Click on the lookup icon to select/modify the appropriate G/L Account 

Branch -  The data on this field can't be modified.

Is Locked? - Click the flag when the Petty Cashier is no longer operational/ being used.

When done click on the UPDATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.
