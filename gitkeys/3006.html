       Performance Period


 

         Use this sub-module to create Performance Periods into the system.

To access this window, click : Human Resources Module -> Navigate -> Setup - > Performance Period.


 

The following screen will open up 


 

Performance Period Fig 1


 

Creating Performance Period :


 

Click on the create button to open the following screen:


 

Performance Period Fig 2


 

On the above screen : 

Provide the  Name.

Click 

to Select Start Date  or alternatively type in the date.

Click 

to Select End Date or alternatively type in the date.


 


 

       Click on Create button to Create the period into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
