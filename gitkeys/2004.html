Cheque Types

Use this sub-module to set up the various Cheque Types.

To access this window, click: Front Office Management - > Setup - > Cheque Types. 

 The following screen will open up: 

Creating A Cheque Type:

Click on the Add New Button, to open the following screen:

On this Create screen, fill in the following:

 

Name - Enter the name of the Cheque Type.

Maturity Days Category - Select the Maturity Days Category of the cheque type from the drop down menu.

Maturity Period - Specify the maturity period of the cheque type is days. 

Charge Recovery Mode - Select the appropriate charge recovery mode from the drop down menu. 

 Applicable Charges Tab: 

Select the appropriate charges for that particular type of cheque. 

Offsetable Loan Products Tab: 

 Check the loan product(s) to be deducted from that cheque if any. 

Offsetable Investment Products : 

 Attach an investment product to that cheque if any by checking against the right ones.. 

When done click on the create button to save the check type into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
