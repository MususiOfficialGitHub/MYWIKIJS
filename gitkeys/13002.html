       POS Terminals


 

Use this sub-module to set up the POS terminals in the system.

These can be in-house or external(agent).

To access this window, click Alternate Channels - > Setup - > POS Terminals. 


 

The following screen will open up:

 POS Terminals Fig 1.

             The already captured POS Terminals will be displayed. 


 

    Creating A POS Terminal : 

          Click on the CREATE Button to open the following screen:

 POS Terminals Fig 2.

    

NOTE : The POS Terminal to be captured must defined as a teller.

     On this screen enter the following details as elaborated below:

Name  - Enter the name of the Terminal.

UniqueIdentifier - Enter the device Id of the POS Terminal.

Teller  - Look up and select the teller(pos terminal in-house/agent) who is to be mapped to the POS Terminal.

Branch  - Look up and select the branch under which the POS belongs.

MSISDN (Mobile Number)  - Enter the MSISDN of the POS Terminal.

          

              Click on the CREATE Button to save the POS Terminal into the system. 

                     

On successful completion you will get a pop up message confirming that the operation was successful.
