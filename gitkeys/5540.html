 Loan Penalties Recovery  

Use this module to perform Intra-Account Loan transfers.                                    

To access this window, click :Credit Management - > Operations - > Recurring Procedures - > Intra-Account Loan Transfers


 

               The following window opens up :

Intra-Account Loan transfer Fig 1. 


 

Select a Loan account that you wish to transfer form the customer lookup as shown below. 

Intra-Account Loan transfer Fig 2. 


 

NOTE : You have to select a loan account first before selecting a transfer account otherwise an error dialog will prompt as shown below


 


 

Finally select the 'transfer to' loan account in the transfer Details as shown below. 

Intra-Account Loan Transfer Fig 3. 


 

         Remarks - remember to fill in the appropriate remarks on the field.

finally Click in the update button to commit the changes.


 

A confirmation dialog box will appear to confirm if you want to continue with the operation.

Intra-Account Loan Transfer Fig 4.

 


 

      

On successful completion you will get a pop up message confirming that the operation was successful.
