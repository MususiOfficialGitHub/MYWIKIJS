Approval

Use this sub-module to approve loans. This should be done by  loan officers only.

This is the immediate loan processing stage after Appraisal.

 

To access this module: Credit Management-> Navigate -> Operations ->Loan Origination -> Approval


 

     The following window will open up: 

 

Loan Approval Fig.1


 

On the above window:

Loan Number - Click on the lookup icon to open the Loan Number Lookup dialog box:

 

Loan Approval Fig.2

              Select the loan to be approved from the list. 

                     

 

NOTE : The loan status must be appraised since the system's operations are sequential. Select the correct loan section, date ranges then refresh. 


 

 The selected customer details will be displayed as below:

 

Loan Approval Fig. 3

The highlighted section above allows the officer approving the loan, to check and confirm the amount applied, what has been recommended by the system plus system remarks,                     amount appraised and appraiser remarks, Loan amount and finally Total Loan+Interest amount for the  customer. 

You can also check for any offsetable loans, loan accounts deductions and repayment schedule  the  loan to be approved. See below


 

 

 Loan Approval Fig 4  : 


 

The lowest section allows the approving officer to update the transaction with either an approve, reject or differ: 

 

Loan Approval Fig 5

     To approve, simply select Approve from the Loaning Remarks Lookup dialog box:

 

 

Loan Approval Fig 6

        Click on the  update button  to complete approve of the Loan.

         

 

On successful completion you will get a pop up message confirming that the operation was successful.
