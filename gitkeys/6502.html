G/L Account Transactions

Use this sub-module to view  transactions for a particular General Ledger account for a selected period.

To access this window, click :Accounts Module - > Navigate - > Operations - > Transaction Journal - > G/L Account Transactions.

The following screen will open up:

 Viewing A G/L Account Transaction Journal

G/L Account - Select a G/L Account name by clicking on the lookup icon to open a G/L Account Lookup dialog Box: 

Select the G/L Account that you want to view.

The type and cost center data fields will be automatically populated once the account is selected

2. Select the date range within which you want to view the transactions.

Filter by Created Date or Value Date ;


 Alternatively, use the search field to retrieve a particular G/L transaction journal.

Click on the refresh button to reload data onto the screen. 

All details pertaining to the transactions within the selected account are displayed. 

The account summary is also displayed on the screen's lower pane.

Viewing A Specific G/L Account Transaction : 

Once data is reloaded onto the screen, a specific g/l  transaction detail can be viewed by: 

1. Identify transaction to be viewed. 

2. Click on the view icon against the transaction to open the following window :  

The G/L Account transaction journals is used to view any  G/L Account affected by a transaction by selecting the appropriate account.
