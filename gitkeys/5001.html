Loan Purposes

These are the reasons for which customers request Loans.

Ensure all possible reasons are exhausted in this sub - module.

To access this module, Click on Credit Management ->Set Up-> Loan Purposes

 The following screen will open up: 

On this window, a new purpose can be captured, and an existing one modified.

Editing Loan Purpose 

To modify a loan purpose, first identify the loan purpose. 

Then click the edit icon  against it to open the following screen: 

Modify the loan purpose accordingly then click on the update button to save the changes into the system. 

Along the same lines, a purpose can be locked if it's no longer in use by checking the checkbox.

On successful completion you will get a pop up message confirming that the operation was successful.
