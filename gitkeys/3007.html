Exit Interview Questions

 Use this sub-module to create Exit Interview Questions into the system.

To access this window, click: Human Resources Module -> Navigate -> Setup - > Exit Interview Questions.

The following screen will open up. 

 All available Questions are displayed. 

 

Creating An Exit Interview Question :

Click on the Add new button to open the following screen:

On the above screen: 

Type in the Question in the field.

Click on Create to Create the Question into the system. 

On successful completion you will get a pop-up message confirming that the operation completed successful.

Editing An   Exit Interview Question :

 Identify the Interview Question and right-click and click on the edit icon against the period to open the following screen:

  

 

On the above screen:

Make changes to the Question fields. 

Check the flag to render the question unusable.

                                                                                                          

 Click Update button to commit the changes into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.
