Inventory Catalog

The catalog sub-module is used view and modify inventories in the system. Each inventory you add has a detailed overview.

To access this sub module click on: Procurement and Assets Management -> Navigate -> Operations -> Inventory -> Catalog

 The following window will open up :

All the available inventories are displayed. 

 Editing An Inventory : 

 Identify the inventory to be edited : 

  Inventory Catalog Figure 2.

             Click on the edit icon 

against the entry to open the following window :


 

Inventory Catalog Figure 3.


 

       On the above screen, make applicable changes to the remarks field. 

                All the other fields cannot be modified. 

Click on the update button to commit the changes to the system. 

             

On successful completion you will get a pop up message confirming that the operation was successful.
