Batch Origination

Use this sub-module to originate membership withdrawals settlement batch.

To access this window, click: Registry Management =>> Operations =>> Membership Withdrawal =>> Settlement =>> Batch Origination

The following Screen will appear:

The list of Settlement Batches is displayed on this window.

 

Creating a Settlement Origination Batch

Click on the Add New button to open the window below:

Branch - This will auto pick the employee's branch by default but using the Lookup icon under the branch field you can choose the branch under which the settlement will be created. 

Category - Proceed to choose the category of the settlement batch using the drop-down menu. The category should match the category of the withdrawal notice to be batched.

Priority - Proceed to choose the priority of the settlement batch using the drop down. This can assist in sorting the order in which the batches will be posted, as a batch with high priority will be posted ahead of a batch with normal priority.

Reference - Proceed to fill in the Reference field with details of the settlement batch to be created. 

Click Create button and on successful completion, you will get a pop-up message confirming that the operation was successful.

 

Editing a Settlement Origination Batch

The batch created must be edited in order to add verified Membership withdrawal Notifications.

Right Click to view the edit button on a batch to add the Members Withdrawal Notifications. :

This opens the membership settlement batch entry window. 

Under the Batch Entries area, Use the Target membership notifications search to select any notifications that exist.

Once you're done click on the Update button. The pop below appears to request to get verification for the batch or not.

Choose yes or No for the above request. Choosing yes Makes a prompt for a Batch verification.

On successful completion you will get a pop up message confirming that the operation was successful.

 

Viewing Membership Withdrawal Settlement Batch

Right click on the Membership window use the view icon against the record you wish to view.

View audit log use this to view the audit log details of the entry selected 

 The window below opens.

The details of the batch Populates in all the available fields. The batch entries that have been created under the batch appear in the Batch entries area.

You can use filters to check the batch entries and use the search filter while looking for a certain entry.
