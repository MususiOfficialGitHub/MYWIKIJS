Template Bases Reports

Use this sub-module to view and export the Templates that are Based on Reports e.g the SASRA reports. 

You can Export these reports to the Original Excel Files to the Template Excel files that need data to be written onto then saving them on the computer. 

This is faster, easier and convenient since the Excel reports will now be generated 

This sub-module can also be used to set update/create report templates. These templates will be used to generate any required report with ease and fast.

For example to create a SASRA Template report you need to have/use the SASRA's Excel templates as a reference.

To access this window click on ; Accounts Module - > Setup - > G/L Accounts - > Template-Based Report.

The following window opens up : 

How to create a report template

1. To create a report template, click on the ADD NEW Button. The following screen will be opened:

On this screen:

Report Template- When creating a first time parent Header Entry, leave this field blank. 

Otherwise when creating a Detail Entry, click on the lookup icon to open a "Report Template Lookup" dialog Box. Select the Parent (Header Entry) under which the Detail Entry will be.

Reference- Enter the appropriate reference of either the Header Entry or the Detail Entry.

Date Filter- Select appropriate date filter as shown below

Cut -Off Date - Select the cut-off date.

Then click on create button  

A) How to Create a Header Entry

1. On the "Parent Template" field, DO NOT Select anything. Leave it BLANK

2. Next in the "Reference", Enter the reference of the Header Entry you want to create for example "FORM1 SASRA1/001: CAPITAL ADEQUACY RETURN".

3. Next Select the date filter

4. Next select the cut-off date

5. To finalize click on the CREATE Button

On successful completion you will get a pop up message confirming that the operation was successful.

6. Go back to the "Report Templates" Tab and click on REFRESH to see if the  Header Entry was created.

B) How to Create a Detail Entry

Using our previously created first Time Header Entry as our Parent Template:

1. Select the Select the "Parent Template" by clicking on the lookup icon to open a "Report Template Lookup" dialogue box. Select the Parent (Header Entry) under which the Detail Entry will be and in this case our Parent Template will be "FORM1 SASRA1/001: CAPITAL ADEQUACY RETURN"

2. Enter the reference of the Detail Entry 

3. Next Select the date filter

4. Next select the cut-off date.

5. To finalize click on the CREATE Button

On successful completion you will get a pop up message confirming that the operation was successful.

6. Go back to the "Report Templates" Tab and click on REFRESH to see if the if the Detail Entry was created.

NOTE:

1. All "DETAIL ENTRIES" are those that will ACCEPT/REQUIRE data input.

2. While all the "HEADER ENTRIES" they DO NOT ACCEPT or REQUIRE data input.

How To Export Data To A Report Template. 

How to select & Export a Report

1. Select the correct "Start date " and " End date "  by clicking on the Calendar Icon. Navigate through the calendar and select the correct Date.

2. Click on the REFRESH Button to load data onto the screen as shown below

3. Click on the view icon of the report template you want to export then it will open the following page    

4. When satisfied, click on the EXPORT Button.The selected report will be exported.

5. Find / Select the Excel Template that you want to load with data. Once you find it click on the OPEN Button (that's on that Open Dialog Box.)

6. Once the exportation process is complete, you will be prompted to SAVE the file, select the folder where you want to save the new file with data.

On successful completion you will get a pop up message confirming that the operation was successful.
