Designations

Use this sub-module to setup new designations.

To access this window, click: Human Resources Module -> Navigate -> Setup - > Designation.

The following screen will open up:                   

All available designations are displayed.

Creating A New Designation.

Click on the add new button to open the following screen: 

Parent Designation - Leave this field blank. 

Name - Type in the name of the parent designation into this field.

Remarks - Type in remarks regarding the parent designation. 

Click Create to capture the designation. 

 

Child Designation:

Parent Designation - Click on the look up icon to open the following designation look up dialog box: 

Name - Type in the name of the child designation into this field.

Remarks - Type in remarks regarding the child designation. 

 

On successful completion you will get a pop-up message confirming that the operation was successful.

Click on the refresh button to see the created designation.

Editing A Designation :

Identify the designation to be edited and right-click and click edit to open the window as shown:

NOTE : The edit feature is used to define the transaction thresholds for the designation. 

 On this window:

Make changes to the name and remarks fields.

Transaction Thresholds:

      i) Pick a transaction type from the drop-down menu: 

Set its corresponding threshold in the threshold field. 

ii) Click on the add button to add the entry to the Transaction Threshold   Preferences pane. 

 iii) Repeat the above steps for each transaction type you want to set the threshold for then add to the pane below. 

  iv) To remove any erroneous entries, select the threshold preference to be deleted then click on the remove button to get rid of the entry.

Click update button to save the changes into the system.

 

 Check if the designation is to be locked.

On successful completion you will get a pop-up message confirming that the operation was successful.
