       Exit Interview Summary


 

          Use this sub-module to view the summary of the exit interview questions responded to by the exiting employee.  

To access this window, click : Human Resources Module -> Navigate -> Operations  - > Employees -> Exit -> Exit Interview -> Summary. 


 

 The following screen will open up:        

                                                                                                                                       

            Exit Interview  Summary Fig 1 


 

                        On the above screen :

Employee -  Click on the employee look up icon to open the following Employee Look Up Dialog Box: 


 

  Exit Interview  Summary Fig 2 


 

          From the above screen, select the correct employee whose exit interview summary is to be viewed. 


 

                             The employee's Department &  Branch  data will be populated to the respective fields. 


 

   Exit Interview Summary Pane: 


 

                   The fields in this pane will be populated with data once the customer is selected. See below: 


 

Exit Interview  Summary Fig 3


 

              Use the scroll bars to view more details about the exit interview summary.
