Data Period Closing

      This module is used  for closing a Data Period. 

       Closing of a data period happens immediately after data processing. 

                                       

NOTE : Remember, once the data period has been closed it cannot be re-opened or amended.

To access this window, click :Credit Management - > Operations - > Data Processing - > Closing


 

The following window will open up:

Data Period Closing Fig 1 


 

How To Close A Data Period: 


 

Data Period - Click on on the Look up icon to open a Data Period Lookup dialog box: 

Data Period Closing Fig  2

          Select the appropriate data period that is to be closed.

                          

NOTE :  The status of the Period has to be OPEN for it to be available for use. 

         Remarks -  The remarks will be populated to the respective field once the data period is selected. 

See below: 

Data Period Closing Fig  3

       Once you have selected the data period, all the entries processed within that data period will be displayed under the data entries pane. 

                         Click on the update Button to close the data period.  

     

On successful completion you will get a pop up message confirming that the operation was successful.
