Catalog

The catalog sub-module is used for viewing and editing of the various Assets an organization has at its disposal. 

To access this sub module click on: Procurement and Assets Management -> Navigate -> Operations -> Assets -> Catalog.

The following screen will open up :

The above list displays all the available assets within the organization. 

 To view more details about a particular asset :

  Identify the asset :

 Click on the view icon against it to open the following window :        

Asset Catalog Fig 3

 

  The three tabs at the lower part of the screen display additional information about the asset. 


 

Asset Catalog Fig 4


 

     The picture tab 

displays the asset's picture. 


 

    The movements  tab 

displays the asset's movement history: 


 

    The depreciations tab 

displays the asset's depreciation history based on the asset's depreciation type.


 


 


 

 Editing An Asset :


 

  Identify the asset to be edited : 


 

Asset Catalog Fig 5


 

                Click on the edit icon 

against the asset to open the following screen :        

                                                                                       

Asset Catalog Fig  6


 

              Name, Accumulated Depreciation, Depreciation Expense G/L Account, Gain Lose G/L Account, G/L Account, Asset Type - data in these fields cannot be modified.


 

        On the above screen, modify the following: 


 

Picture - click on the picture look up dialog box to open the windows explorer dialog box:  


 

Asset Catalog Fig  7


 

           Search for the asset's picture saved on you computer then double click on it to upload to the system. 

Make necessary amendments to the Serial Number, Manufacturer, Model & Tag Number fields.

The Location & Purchase Price  fields cannot be modified. 

Make necessary changes to Opening Book Value, Useful life, Residual value & Remarks fields.

Opening book value -  This is  the value of an asset according to its balance sheet account balance, based on the original cost of the asset less any depreciation,                                        amortization or impairment costs made against the asset.

Useful Life - This is the period of time (or total amount of activity) for which an asset will be economically feasible for use in a business.

Residual Value - This refers to the remaining value of an asset after it has been fully depreciated. 

                           Type in the residual value of the asset into this field. 

Remarks - Type in the appropriate remarks concerning the action being carried out i to this field.


 

 Note: 

 Once an asset is fully depreciated, it is due for disposal.

 flag is used to earmark an asset for demolition. 

   To earmark an asset for demolition, simply check the disposal flag   

.


 

            Update to save the changes into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
