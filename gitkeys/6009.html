Charges

Use this sub module to setup (create and edit) the Charges.

A Charge refers to a certain amount that the organization charges for various transactions and services offered.

To access this sub-module click; Accounts Management - > Setup - > Levies & Charges - > Charges

 The following window opens up:

  Creating A Charge:

 Click on the ADD NEW button to open the following window: 

On the above screen: 

Name - Enter the name of the charge being created into this field. 

Maximum charge - Type in the maximum charge to be associated with this charge. 

Rounding Type - Select the rounding type to be associated with this charge from the drop down menu: 

NOTE: On this screen we have three Buttons: i) Charge Splitting ii) Tiers & iii) Levies.

i) Charge Splitting Button

 

 Use this button to split the charge into one or different G/L accounts. 

On this pane:

G/L Account - Click on the lookup icon to open the G/L Account Lookup dialog box. Select the appropriate G/L account for that Charge.

Name - Enter a brief description of that Charge

Percentage - Enter the percentage of the charge that will go to the selected G/L account.

Click on the add button to append the entry to the Charge Splitting pane just below: 

All the charge amount  will go to the cash withdrawal G/L account. (If the charge amount is not being split i.e if  its only going to one account, enter 100% as the  percentage.)

ii) Tiers Button: 

On this Button:

Range (Lower Limit) - Enter the Lower Limit of the scale entry. 

Charge Type - Click on the drop down arrow and select the correct type: percentage or fixed amount.

Charge - If you select the charge type to be percentage, then Enter the percentage that will be  used for the Charge.

If you select the charge type to be Fixed Amount, then enter a fixed amount that will be charged.

 Editing A Charge:

1. Identify the Charge to be edited: Click on the edit icon (against that Charge) to open the following window: 

On this edit screen:

Make the necessary changes to the relevant fields appropriately - name, maximum charge, rounding type, charge splitting, tiers and levies tabs.

Click on the update button to effect the changes. 

On successful completion you will get a pop up message confirming that the operation was successful.
