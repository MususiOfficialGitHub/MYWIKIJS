Verification

Use this module to verify a registered account closure request.

To Access this sub module, click On Front Office Management->Operations ->Account Closure ->Verification.

The following window opens up: 

On the above window:

Full Account Number - Click on the look up icon to open the Customer Account Closure Request Lookup dialog box:    

The status of the account closure request to be verified should be approved.
Any other status will not be able to pick.

Select the correct date range.

Narrow down the filter using the search strings available, then select the entry to be verified. 

Data will be populated to the relevant fields.

Cross check through the other tabs to view more details about the customer.

Audit Remarks - Enter the audit remarks appropriately.

 

Select the verify action from the drop-down menu then click on the update button effect the verification.

On successful completion you will get a pop-up message confirming that the operation was successful.

 

The Account Closure can now be paid over the counter by the teller via; Teller> Sundry Receipts/Payments>Account Closure Payment
