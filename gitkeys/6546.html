Catalogue

Use this sub-module to view the mobile to bank transactions.

To access this window, click: Accounts Management- > Operations - > Mobile To Bank - > Catalog


The following screen will open up : 

The available requests are displayed.  

 Select a date range within which you want to view transactions then refresh to reload data onto the screen.
