Catalog

Use this sub-module to view the detail entries of a particular Data Period. 

To access this window, click :Credit Management - > Operations - > Data Processing - > Catalog


 

   The following screen will open up: 

 Data Period Catalog Fig 1

             

           To View A Data Period:

 Data Period - Click on the lookup icon to open a Data Period Lookup dialog box:

 Data Period Catalog Fig 2 

                       Specify the start and end dates within which you intend to view data period details, select the status from the drop down menu then refresh. 

                       Alternatively, type in a keyword into the search field then refresh to reload the data onto the system. 

                        Select the desired data period.

NOTE : This sub module is used to view any status of the data periods -  closed, open or suspended. 

                          No actions can be carried out on any entry in this sub module. 

    The detail entries of the selected data period will be displayed as below:

Data Period Catalog Fig 3

             Click on the items per page drop down and select the number of items to view per page, then refresh:
