Loan Disbursement Modes

Use this sub module to capture the disbursement modes that will be used during loan processing

To access this module: Credit Management -> Navigate -> Setup -> Loan Disbursement Modes

The following screen will open up: 

 

Creating  Loan Disbursement Mode 

To  create a loan mode, click on the Add New button to open the following window: 

 On the above screen: 

Name - Type the name of the disbursement mode.

Disbursement Category:  Select the disbursement category

Saving Product : Look up and select the savings product

Applicable Charges: Tick the applicable charge as shown below

Click on the create button to save the remark.

Tick the flag if the disbursement mode should recover arrears  

Tick this flag to lock the disbursement mode

On successful completion you will get a pop up message confirming that the operation was successful.

 

Editing A  Mode.

Identify the Loaning Disbursement Mode to be edited. 

 Click on the edit Icon against the remark.

The following screen will open up: 

 

Make the necessary changes to the mode then click on the update button to save the changes into the system. 

A  remark can also be locked if it's no longer in use or for any other valid reason as far as editing is concerned.

 To lock, click in this checkbox :  

 To unlock, clear the check box.

On successful completion you will get a pop up message confirming that the operation was successful.
