Employee Self Appraisal.


 

           Use this sub module  to perform self appraisals whenever an appraisal period is created. 

To access this window, click :Human Resources Module -> Navigate -> Operations -> Performance management ->Self Appraisal. 


 

          The following screen will open up: 


 

Employee Self Appraisal Fig 1


 

        On the above screen: 

Employee:  The system will default to the user who is currently logged in. 

               

REMEMBER: This module is for employees self- appraisal. 

                                                  The employee field will always therefore populate the name of the currently logged employee. 


 

Appraisal Period : The system will automatically pick the appraisal period which is active at the moment.

                              

     Appraisal Questions: 


 

 Question: The appraisal questions will be populated according to the order in which they were created. 

                To access the next question in the appraisal process, click on the next button on the numbering bar, to the bottom right. See below: 


 

             

             Employee  Self Appraisal Fig 2


 

Answer :  To provide an answer to the appraisal question, click on the up down scroll keys to set a figure. 

               Type in an answer if the answer type is text. 


 

 

NOTE: The appraisal factor is rated between 0 - 10. Therefore you'll have to rate yourself within the enlisted range. 

             For every question answered, you have to click on the save button to save the information into the system before proceeding to answer the succeeding questions.

            For answers whose type is text, a text box will be presented wherein you'll type your feed back. 

On successful completion you will get a pop up message confirming that the operation was successful.
