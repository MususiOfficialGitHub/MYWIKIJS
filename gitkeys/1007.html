Numbering Series

Use this sub module to capture numbering series. This is a Numbering system for members according to member number per membership class or account number per branch.

Click on System Administration from the menu items; System Administration -->Setup--> Numbering Series.

The following screen opens up as the available numbering series are displayed: 

 

Creating a Numbering Series

Click the Add New icon to open the following window:

On the above screen:

Type in the window fill in the respective fields.

Name: Enter the name of the numbering series e.g. member numbering, account numbering.

First Number: enter the first number to be generated in the series that will be assigned to a member.

Next Number: enter next number that will follow the first number.

Last Number: enter last number in the series.

Padding:  enter the padding, i.e. number of characters the series generated will have. 

Prefix: enter prefix i.e. this will be shown at the beginning of the generated number e.g. Ac or Mn.

Suffix: enter suffix i.e. this will be shown at the end of the generated number.
 

Click on Create once done.

On successful completion you will get a pop-up message confirming that the operation was successful 

 

Editing a Numbering Series

 Identify the numbering series to be edited right click on it then click edit:

The following screen will pop up:

On the above screen, edit the necessary fields accordingly.

Is Locked? - This is a flag that renders the currency unusable when ticked.

Generate series? -  Tick this flag to generate sequence series which will populate the view screen shown below from to generate the numbering series.
