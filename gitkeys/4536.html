Recall

This sub-module is used to recall files which haven't been received by the destination department back to the Registry Department.

Only users in the registry department can recall files back to the Registry.

To access this module, click :Registry Management =>> Navigate =>> Operations =>> File Tracking =>> Recall

The following window will open up: 

How To Recall A File

Search and select the file to be recalled. Search filters like account number, first name, identity card number, last name can be used to retrieve specific entries. See below: 

Select the file by checking (ticking) the check box against it: 

Click on RECALL Button to complete the process.

On successful completion you will get a pop up message confirming that the operation was successful.
