Bank Linkages

Use this sub-module to set up the bank-linkages i.e to link a bank to a Branch and to its specific G/L account.

To access this window, click:Accounts Management - > Setup - > Miscellaneous -> Bank Linkages

The following screen will open up: 

 Creating A Bank Linkage: 

Click on the ADD NEW  Button, to open the following screen:

On the above screen: 

Bank Name - Click on the lookup icon to open Bank Branch Lookup dialog box: 

SELECT the Bank and then SELECT the branch from the lower pane of the screen. 

Bank Branch Name - This Data will be automatically populated to this field when the bank & branch is selected. 

Bank Account Number - Key in the correct bank account number of the bank linkage being created. 

G/L Account - Click on the lookup icon to open G/L Account Lookup dialog box: 

Select the appropriate Bank G/L Account.

Branch Name - Click on the lookup icon to open Branch Lookup dialog box:

Select the associated Branch name of the G/L Account. 

Remarks - Enter the remarks appropriately describing the linkage being created.  

When done click on the CREATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.

Editing A Branch Linkage: 

First Identify the Bank Linkage to be edited: 

Then click on the edit icon against it to open the following window:

On the above edit screen: 

Make necessary changes to the applicable fields - Bank Name, Bank Branch Name, Bank Account Number, G/L Account, Branch Name 

Remarks - Key in the new remarks regarding the linkage. 

Check on the flag if linkage is not in use.

When done click on the UPDATE Button to complete this task. 

On successful completion you will get a pop up message confirming that the operation was successful.
