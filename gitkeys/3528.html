Salary Groups 


 

           Use this sub module to  create Job Groups and their corresponding salary ranges according to which employees will be paid. 

To access this module click on Human Resource Module -> Navigate ->Operations -> Salary -> Salary Groups


 

            The following screen will open up :


 

Salary Groups Fig 1.


 

                 The available salary groups are displayed. 

         


 

Creating A Salary Group :


 

      Click on the create button and the following screen will open up :


 

Salary Groups Fig 2.


 

       Name -  Provide an appropriate name for the salary group ; Salary Group A, Salary Group B....


 


 

            Salary Group Entry Details :


 

Salary Groups Fig 3.


 

         Salary Head - Pick the appropriate salary head from the Salary Head Look Up dialog box: 


 

Salary Groups Fig 4.


 

          Pick a value type from the drop down menu, input the corresponding value for the salary head, define the salary head's minimum value then  pick an appropriate rounding type from the drop down 


 

Salary Groups Fig 5.


 


 

                  Click add to commit the entry to the Salary Group Entries pane just below: 


 

Salary Groups Fig 6.


 

             

NOTE: The above process is to be carried for every salary head that is to be included in the salary group being captured . 


 

                                For an erroneous entry, select the entry then click remove to effect its removal from the pane.


 

                                           Click create to commit the salary group into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.


 


 

                                   


 


 


 

     Editing A Salary Group :


 

              Identify the salary group to edit.


 

                           Click on the edit icon 

against the salary group to open the following screen :


 

Salary Groups Fig 7.


 

             Make appropriate changes to the Name field. 


 

      Add to or remove salary group entries from the Salary Group Entries Pane where applicable. 


 

                         Click update to commit the changes to the system.

On successful completion you will get a pop up message confirming that the operation was successful.
