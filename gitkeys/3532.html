Pay Slips


 

             Use this module to view employee pay slips.        

To access this module click on Human Resource Module -> Navigate -> Operations -> Salary -> Pay Slips. 


 

The following screen opens up :


 

Payslips Fig 1


 

            Salary Period - Select the salary period from the Salary Period Look Up dialog box: 


 

Payslips Fig 2


 

           All the salary cards processed in that salary period will be displayed, together with their details : 


 

Payslips Fig 3


 

                   Select a salary card to view its payslip details, which will be populated to the Pay Slip Entries Pane :

        For instance card number 000017: 


 

Payslips Fig 4


 

Use scroll bar to check details in the Pay slip entries section shown above
