Receive

This sub-module is used to receive a dispatched file. 

Vanguard users in the department where the file has been dispatched to are the only ones who can receive these files.

To access this sub module, click: Registry Management =>> Navigate =>> Operations =>> File Tracking =>> Receive

The following window will open up: 

Click on the "REFRESH" Button to reload the data on to the screen.

 If there are any files dispatched and ready to be received, they'll be displayed as shown below:

All dispatched files to that respective department will be displayed.  Search filters - account number, first name, serial  number, last name - can also be used to retrieve specific files

To receive files into the department, select the specific file(s) to receive by selecting the check box against them: 

The File Movement History pane will in turn display the source of the file when the file is selected: 

The destination department, carrier, sender and dispatch dates will also be displayed on the file movement history pane. 

Click on the receive button to receive the files into the department. 

On successful completion you will get a pop up message confirming that the operation was successful.
