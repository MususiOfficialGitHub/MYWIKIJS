Disciplinary Cases

Use this sub module to capture employee disciplinary cases into the system for future reference.

Such cases include, but are not limited to violation of ethics, misconduct and anomalous behavior. 

NOTE: The documents uploaded here must be SCANNED COPIES in PDF format ONLY.

To access this module, click on: Human Resources Module -> Navigate ->Operations-> Employees -> Disciplinary Cases.

The following screen will open up: 

 The available cases are displayed on the screen:

 

Creating A New Disciplinary Case 

Click on the Add new button to open the following screen: 

 

Employee - Pick the employee to be affected from the dialog box:

 

Incident Date - select the date on which the incident occurred.

Disciplinary Type - Pick the disciplinary type from the drop-down menu:

Document - click on the look up icon to open the following windows explorer dialog box: 

Search for the document to be uploaded then double click on it once located to upload. 

Provide an appropriate Title, Description & Remarks regarding the case. 

Refer to the figure below: 

 

 Click create to commit the case to the system. 

On successful completion you will get a pop up message confirming that the operation was successful.

 

Editing A Disciplinary Case :

Identify the disciplinary case  then  right-click on the edit icon against the case to open the following screen: 

 On the above screen make changes to the case's Incident Date, Disciplinary Type, Document, Title, Description &  Remarks fields.

  Click update to commit the changes into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.
