Cash Management Requests

This sub module is used to view all Treasury Cash Transfer Requests in all status (Pending, Authorized, Posted, Rejected).

To access this module, Click on Front-Office Management -> Operations -> Treasury -> Cash Management Requests. 

Select the date range as of when you wish to view the transactions.

To view more records on the page, adjust the Records Per Page to view up to 1000 records.

Use the Status filter to search for treasury management requests that are either Pending, Authorized, Posted or Rejected.

Use the Date Range filter to check records withing a certain date range dating back from 15 days to 6 months or use a Custom Date Range to set a date range that for previous years.

Use the Search filter to search for records by their description, transaction type etc.
