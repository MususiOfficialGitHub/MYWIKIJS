Wire Transfer Types

Use this sub-module to set up the wire transfer types products.

To access this window, click :Accounts Management - > Setup - >Miscellaneous -> Wire Transfer Types. 

The following screen will open up :


Creating A Wire Transfer Type : 

Click on the ADD NEW Button to open the following screen: 

On the above screen: 

Name  - Enter the name of wired transfer type being created. 

G/L Account  - Click on the look up icon to open the following G/L Account Lookup Dialog Box : 

 Select an appropriate account to be mapped onto the wire transfer type being created. 

Transaction Ownership  - Select the appropriate transaction ownership to be enforced on the wire transfer type being created from the drop down menu. 

Applicable Charges  - Select the appropriate applicable charges to be enforced on  the wire transfer type being created:

 NOTE: If the IS LOCKED flag is checked, the transfer will not be available for use. 

 Click on the create button to save the changes into the system. 

 On successful completion you will get a pop up message confirming that the operation was successful.

 

                            

   Editing A Wire Transfer Type : 

First Identify the wire transfer type to be edited : 

Click on the edit icon against it to open the following screen: 

On the above screen, modify the Name, G/L Account, Transaction Ownership & Applicable Charges fields accordingly. 

Click on the update button to save the changes into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
