Monthly Account Balances

Use this sub module to compute monthly account balances - for savings products that yield interest on a monthly basis

To access this window, click :Accounts Module - > Operations - >Recurring Procedures - > Monthly Account Balances

The following screen opens up :

Monthly Account Balances Fig 1.

All the records are displayed on the screen as shown above. 


 

 Create a Monthly Account Balance :

Click on 

to open the following window:

Monthly Account Balances Fig 2


 

Click on the lookup Icon 

to select an active  Posting Period

       Click on the date icon 

to select Start and end Dates (alternatively you can manually type the date in the text field)

       Balance Statistic: select a balance type from the drop down list shown below

Monthly Account Balances Fig 3.

Lowest Balance - Select this to select products with the lowest balances 

Average Balance - Select this to Select products with  Average balances 

Closing Balance - Select this to Select products with closing balances 

Daily Balance - Select this to Select products with daily balances 

Highest Balance - Select this to Select products with the Highest balances 

Opening Balance - Select this to Select products with opening balances 

Operating Balance - Select this to Select products with operating balances 


 

       Navigate through the highlighted tabs below to select between Savings and investment products as shown below:

Monthly Account Balances Fig 4


 

Use the toggle selection flag  

 if all the available Products are to be selected. 

Click on the Create. 


 

   

On successful completion you will get a pop up message confirming that the operation was successful.
