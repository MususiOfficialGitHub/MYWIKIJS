Withdrawal settlement modes

Use this sub-module to set up the withdrawal settlement modes.

To access this window, click :Accounts Management - > Setup - >Miscellaneous -> Withdrawal settlement modes. 

The following screen will open up : 

The already captured withdrawal settlement modes  will be displayed. 

  Creating a withdrawal settlement mode : 

To Create an Insurance Company, click on the ADD NEW  Button to open  following screen :

On the above window, fill in the details: 

Name -  Enter the name of the Insurance Company 

Consolidation product - lookup to pick the product the following window will open;

Disbursement Product - lookup for the disbursement product the following window will open

select the applicable charge on the applicable charges pane below by ticking

 When done filling in all the data, click on the CREATE Button.

On successful completion you will get a pop up message confirming that the operation was successful.

 

  Editing withdrawal settlement mode : 

First Identify the withdrawal settlement mode to be edited : 

Click on the edit icon against it to open the following screen: 

select to make the settlement default is locked flag if no longer in use

On the above screen, make the necessary changes to the fields. 

Click on the update button to save the changes into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
