Events

 This module is for record employee Engagements at any given point for instance auditor meeting.

To access this module, click on: Human Resources Module -> Navigate ->Operations-> Employees -> Events.

The following screen will open up:

The available employee referrals are displayed. 

Creating An Event 

Click on the Add new button to open the following screen:

 Employee - Click to lookup an Employee from the Employee dialog box to open the shown window.

Title- specify event title.

Description - give a brief description of the event.

Type - Select type from the drop down list as shown below

Click 

to select the Start & End Date or alternatively type in the date

Define the start & end time  

Specify Location & Remarks respectively.

Click create to capture the data into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.

Editing An Event :

Identify and right-click on the Event to be edited, Click on the edit icon against the record to open the following window :

Make necessary changes to the employee's details

Check 

if event is no longer Needed

 Update to save the changes. 

On successful completion you will get a pop-up message confirming that the operation was successful.
