Verification

Use this sub-module to verify membership termination requests.

To access this window, click: Registry Management =>> Operations =>> Membership Withdrawal =>> Verification

The following screen will open up: 

Carry out the following on the above screen: 

Customer - Click on the look up icon to view the approved withdrawal notifications. Select the customer whose request is to be verified. 

NOTE: The status of the request to be verified must be approved. Otherwise, the operation won't be successful. 

 

Data will be automatically populated to the other fields when the customer is selected.

Settlement Batch Number - Select a settlement batch which should have been created prior to this operation, once the notification is verified, it will be inserted into the batch automatically.

Note: The termination category of the notification should match the settlement batch category, i.e. a deceased notification should only be batched in a deceased settlement batch. Otherwise, the following error will be encountered:

Select Mode - This is a mode defined on Accounts Module where you can set the charges associated with different types of settlement modes, e.g. charging consolidated refundable amount or not and whether to suppress netting off loans.
Select a Membership Withdrawal Settlement Mode from the Settlement Mode lookup window.

Select any of the below tabs to view the products and balances of the customer, together with loans guaranteed, if any. 

Loans Account - Displays a member's loans accounts and the balances.

Savings Account - Shows you a summary of the member's savings account.

Investment Accounts - Displays a summary of a members investment accounts.

Loans Guaranteed - Displays loans a member has guaranteed.

Audit Remarks - Enter the audit comments into the audit remarks field. 

Select on the verification action or defer to verify the withdrawal notice.

Click on the update button to save the information into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.
