Origination

This module is used to originate the payment of interest accrued on a super savings product. 

To access this sub module, click on: Front Office Module Management-> Operations -> Super Saver Payables -> Origination.

The following screen will open up:

To originate a liquidation entry, click on the Add New button to the bottom right of the screen to open the following window: 

On the above screen, fill the following: 

Full Account Number - click on the look up icon to open the following Customer Account Look Up dialog box: 

From the above list, select the customer whose super saver interest is to be paid. 
Alternatively, type in a search filter then refresh to retrieve the specific customer whose payment is to be originated. 
Once the data is retrieved, select the super deposits account option.  

Account Status - data will be automatically populated to this field when the account number is selected. 

Account Remarks - data will be automatically populated to this field when the account number is selected. 

Product Name - data will be automatically populated to this field when the account number is selected. 

Customer Name - data will be automatically populated to this field when the account number is selected. 

Employer Name - data will be automatically populated to this field when the account number is selected. 

Customer Type - data will be automatically populated to this field when the account number is selected. 

Customer Serial Number - data will be automatically populated to this field when the account number is selected. 

Identification Number - data will be automatically populated to this field when the account number is selected. 

Member Number - data will be automatically populated to this field when the account number is selected. 

Account Number - data will be automatically populated to this field when the account number is selected. 

Personal File Number - data will be automatically populated to this field when the account number is selected. 

Book Balance - data will be automatically populated to this field when the account number is selected. 

Interest - data will be automatically populated to this field when the account number is selected. 

Withholding Tax Amount - data will be automatically populated to this field when the account number is selected. 

Remarks - Type in the appropriate origination remarks into this field. 

 

Once done, click on the create button to save the data into the system. 

On successful completion you will get a pop-up message confirming that the operation was successful.
