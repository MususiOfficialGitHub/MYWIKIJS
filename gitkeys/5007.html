Data Attachment Remarks

These are remarks that will be attached during data processing.

To access this module: Credit Management -> Setup -> Data Attachment Remarks.

The following screen will open up: 


 

Creating Data Attachment Remarks.

To  create a data attachment remark, click on the Add New button to open the following window: 

On the above screen: 

Name - Type in the remark being captured into this field.

Is Locked? - Select this flag to lock the remark so that it may no longer be used.

 

Click on the create button to save the remark.

On successful completion you will get a pop up message confirming that the operation was successful.

 

 

Editing A Data Attachment Remark.

Identify the Loaning Remark to be edited. 

Search for the remark and right click on the remark you wish to edit. The following screen will open up: 

Make the necessary changes to the remark on the Name field. 

Is Locked? - Select this flag to lock the remark so that it may no longer be used.

 

Click on the update button to save the changes into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
