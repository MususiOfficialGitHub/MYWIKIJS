 Loan Aging Spools.  

     Use this sub-module to check loans performance in the various loan batches.

                                                     

To access this window, click :Credit Management - > Operations - > Recurring Procedures - > Loan Aging Spools. 


 

               The following window opens up : 

Loan Aging Spools Fig 1. 

          

To check a customers loan performance, click on the view button against a batch record.  

All loans under the batches are usually displayed as shown below. 

Loan Aging spools Fig 2. 


 

You can use filter the records by use the filter Drop-downs as you search for records inside the batch


 


 

     Creating Loan aging spools: 

Click on the Create button on the default loan spool page to create a batch. 

The page should be appear as below.

  Loan aging spools Fig 4.

Define the appropriate Reference for the batch and the cutoff date and set the date filter then click on create to finish the operation


 

     

On successful completion you will get a pop up message confirming that the operation was successful.


 

                               


 

Resetting Loans in loan spools: 

 Click on the reset 

icon against a loan record in the batch to effect any changes on loan parameters that may have been altered. 

This can also be done using the reset button 

to update more than one record at once. 

NOTE : Resetting will only make changes if parameters altered on the said records cause a change on the overall classification of a loan record.  E.g A loan classified as Substandard before reset might be found on the Watch class if the changes on parameters compute to "poorer" repayments on the loan


 

      

On successful completion you will get a pop up message confirming that the operation was successful.
