Conditional Lending

This sub module is used to set up the type of loan that only a specific group of  people are eligible to get/ apply for.

For instance, staff loan - only staff members can be able to get that type of loan.

To access this module click on  

Registry Management =>> Navigate =>> Operations =>> Members Management =>> Conditional lending

The following window will open up:

The already captured conditional lending s are displayed on the screen as above. 

 

Creating A Conditional Lending:

Click on the add button to capture a new conditional lending into the system.

The following screen opens up:

On the above window, fill in the following: 

Name - Enter the name of the type of conditional lending being created. 

Loan Product - Click on the loan product search icon to open the Loan Product Look Up dialog box:

Select the loan product to be tagged to the lending being set up. 

NOTE :  Remember to select a Filter in the Product Category  under loan products from the menu 1 circled in red when searching for a loan product.

NOTE :  Remember to select between FOSA  and  BOSA loan products from the menu 2 circled in red when searching for a loan product.

REMEMBER : A custom loan product to be attached to the conditional lending might not be among the already created loan products. 

This results will need to have the loan product first created before it can be attached to the conditional lending. 

To create a new loan product, refer to Loans under financial products in the set up section of the accounts management.

Target Customer Details Pane:  

On the above pane, do the following: 

Customer - Click on the customer search icon to open the Customer Look Up dialog box: 

Search for then select  the customers you want to add to the conditional lending list.

Use the filters to determine the order of which to group and display the list of customers


 

The retrieval process can be further narrowed down using account number, first name, last name, identity card number etc.

The customer's employer, station, account number, membership number and personal file number will be automatically populated to the respective fields once the customer   is selected. 

Remarks - Provide appropriate remarks in the remarks field. 

Click on the add button to add the customer to the target customer list: 

NOTE : The above steps have to be carried out on each customer who is to be added to the conditional lending list. 

Along the same lines, an erroneously captured customer can be removed from the list by selecting the customer then clicking on the 

remove button to effect the removal:

Click on the create button to save the information into the system once done. 

On successful completion you will get a pop up message confirming that the operation was successful.


 Editing A Conditional Lending :

  Modifying a conditional lending in this system is normally associated with, but not limited to:

  • Changing Of The Name   and
  • Addition/ Removal Of Target Customers.

To edit a conditional lending: 

Identify the entry to be modified and right click on the selected entry:

Then click on the edit icon against it to open the following window: 

Make the necessary changes to the name. 

 Add or remove customers accordingly then click on the update button to save the changes into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
