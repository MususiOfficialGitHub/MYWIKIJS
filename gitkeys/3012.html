       Performance Measure


 

         Use this sub-module to create Performance Measure into the system.

To access this window, click : Human Resources Module -> Navigate -> Setup - > Performance Measure.


 

The following screen will open up :


 

Performance measure Fig 1


 

Creating Performance Measure :


 

Click on the create button to open the following screen:


 

Performance measure fig 2


 

On the above screen : 


 

Click 

to look up Department, Performance objectives & Appraisal period.

Provide the  Name & Target.

Specify weight.

Select Answer type in the drop down as shown below(Select Number when answer is a rating eg. 1-10, select Text  if answer is a typed response).


 

Performance measure fig 3


 

       Click on Create button to Create the measure into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
