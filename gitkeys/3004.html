Employee Types

Use this sub-module to create employee types into the system. 

To access this window, click: Human Resources Module -> Navigate -> Setup - > Employee Types. 

The following screen will be open up:

  All available employee types are displayed. 


 Creating A New Employee Type 

Click on the Add new button to open the following screen:                   

 

On the above screen.

Name - Type the name of employee type being created. 

G/L Account: Click on the G/L Account look up icon 

and select the g/l account to be attached to the employee type on the next window as shown.        

Category: Select the employee category from the drop-down menu. 

Create the employee type into the system.  

On successful completion you will get a pop-up message confirming that the operation was successful.

               

Editing An  Employee Type 

Identify the employee type to be edited and right-click to the window shown then click on the edit icon.

against the employee type to open the following window:

  Make applicable changes to the Name, G/L Account & Category fields and check Offsetable Products from the tabs highlighted above. 

click on update button update the changes into the system. 

 

On successful completion you will get a pop-up message confirming that the operation was successful.
