System Transactions

Use this sub-module to view all system transactions for a particular  selected period.

To access this window, click :Accounts Module - > Navigate - > Operations - > Transaction Journal - > System Transactions.

The following window will open up: 

The transactions are filtered and displayed according to the selected date range.

 Viewing System Transactions Journals : 

Select the date ranges from the calendar icons - start date & end date by clicking CUSTOM. 

See below: 

Select the appropriate search parameter to use to find the transactions from the drop down menu : 

Type in a keyword into the search field, with particular reference to the selected search filter. 

Click on the refresh button and data will be populated to the screen as below: 

A specific entry can also be retrieved by selecting the appropriate date, filter, keyword then refreshing. 

Viewing A Specific System Transaction : 

Once data is reloaded onto the screen, a specific system transaction detail can be viewed.

Identify transaction, click on the view Icon against the transaction to open the following window :  

On the above screen, every detail of the transaction is displayed - Transaction code, branch , posting period down to the integrity hash.  

The respective journal entries for the selected transaction are also displayed on the lower part of the screen : 

Along the same lines, the General Ledger accounts affected by the transaction can also be viewed:

This is achieved by clicking on the view icon against a transaction; which will then present the following window :

On the above screen, the G/L Accounts affected by the transaction, the Account Type and the Cost Center will be displayed on the upper section of the screen. 

More details are displayed just below (see the above picture) 

The account summary is also displayed on the screen's lower pane: 

The G/L Account Look up icon can also be used to view any other G/L Account affected by a transaction by selecting the appropriate account.
