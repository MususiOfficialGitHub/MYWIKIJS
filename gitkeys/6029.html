Transaction References

Use this sub-module to set up transaction references.

To access this window, click :Accounts Management - > Setup - >Miscellaneous -> Transaction References. 

The following screen will open up : 

The already captured references  will be displayed. 

 Creating a transaction reference : 

 Click on the ADD NEW  Button to open  following screen :

On the above window, fill in the details: 

Name - Pre-define a reference here

click on the CREATE Button.

On successful completion you will get a pop up message confirming that the operation was successful.

  

                                      

   Editing transaction reference : 

 First Identify the entry to be edited : 

Click on the edit icon against it to open the following screen: 

select IS LOCKED ;flag to prevent the entry from being used.

On the above screen, make the necessary changes to the fields. 

Click on the update button to save the changes into the system. 

On successful completion you will get a pop up message confirming that the operation was successful.
