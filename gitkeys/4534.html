Single-Destination

This sub-module is for dispatching a files to a single destination for further processing.

To access this sub module:

Registry Management =>> Navigate =>> Operations =>> File Tracking =>> Dispatch =>> Single-Destination.

The following screen will open up: 

Destination - click on the lookup icon to open the following Department Lookup dialog box:

Select the Destination Department.

Remarks - type remarks regarding the file movement into this field.

Carrier -  Type in  the name of the file's carrier  into this field. 

Click on the refresh button to load the customer register and select the customers whose files are to be dispatched.

You can search for a specific customer using the highlighted filters below: 

Select the file to be dispatched by checking (ticking) the check box against it: 

The File Movement History pane will in turn display the source, destination and recipient of the file when the file is selected: 

 Click on the dispatch button to complete the single-destination file movement.  

On successful completion you will get a pop up message confirming that the operation was successful.
