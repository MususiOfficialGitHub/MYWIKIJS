Applications 


 

This sub module is used to view all the loans that have been processed in the system within a particular period of time.


 

All the Loan statuses are also displayed  using this option .i.e approved,disbursed,audited,or rejected.


 

To access this sub module Click on Credit Management -> Navigate -> Operations - > Loaning Origination - > Applications


 

The following screen will open up:

 

Loan Applications Fig 1 


 


 

How To View Loan Applications:


 

   Select the desired loan section from the drop down: BOSA or FOSA.

   Select the start date and end dates from the calendar icons. 

   Narrow down the search filter using the account number, serial number, last name etc..         

 Once done, click on the refresh button to reload data onto the screen as shown below: 

 

Loan Applications Fig  2
