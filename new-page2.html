Forex Rates

Use this sub module to capture Foreign Exchange rates.

Click on System Administration from the menu items; System Administration -->Setup--> Forex Rates.

The following screen opens up as the available Forex Rates are displayed: 

 

Creating a Forex Rate

Click the Add New icon to open the following window:

On the above screen:

Base Currency - Click on the look up icon as shown in the picture below to select base Currency.

Quote Currency - Click on the look up icon as shown in the picture below to select quote currency.

Effective Date - Select date.

Average Rate - Define the average rate.

 

Once done click on create and the following screen should pop up once successful.
